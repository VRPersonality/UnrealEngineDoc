17/**
18 * IGeometryProcessing_ApproximateActors is a generic high-level interface to a function/system that
19 * outputs a set of new Mesh assets that approximate a set of input Actors.
20 * This can be used to provide LOD-generation implementations for systems like ProxyLOD and HLOD.
21 *
22 * This is an IModularFeature, and so clients can query/enumerate the available ApproximateActors implementations
23 * based on the ::GetModularFeatureName(). However, the preferred way is to use code like the following:
24 *
25 * IGeometryProcessingInterfacesModule& GeomProcInterfaces = FModuleManager::Get().LoadModuleChecked<IGeometryProcessingInterfacesModule>("GeometryProcessingInterfaces");
26 * IGeometryProcessing_ApproximateActors* ApproxActorsAPI = GeomProcInterfaces.GetApproximateActorsImplementation();
27 *
28 * This will automatically determine which available implementation to use (based on any applicable config settings/etc),
29 * and cache the result of that decision.
30 */
