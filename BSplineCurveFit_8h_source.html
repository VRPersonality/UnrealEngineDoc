11/**
12* Compute the control points that result from a best fit of the provided 2d data points when using b-splines of the specified degree.
13*
14* This fit assumes the sample points are evenly spaced in 't' over the interval [0, 1].
15* and the knot vector has unique knots at j / (NumControlPoints - SplineDegree) for j in [0, (NumControlPoints - Degree)].
16* the knots at 0 and 1 each have multiplicity SplineDegree + 1, while the internal knots just have multiplicity 1.
17*
18* @param DataPoints - an array of 2d points the b-spline will attempt to fit
19* @param SplineDegree - the degree of the underling b-spline. Must be greater that 0.
20* @param NumControlPoints - requested number of control points solved for.
21* Note: requires SplineDegree < NumControlPoints <= DataPoints.Num()
22* @param ControlPointsOut - on successful return, an array of 2d control points of the specified length ( on failure the array is unchanged ).
23*
24* @return true on success.
25*
26*/
27bool GEOMETRYALGORITHMS_API BSplineCurveFit(const TArray<FVector2f>& DataPoints, const int32 SplineDegree, const int32 NumControlPoints, TArray<FVector2f>& ControlPointsOut);
