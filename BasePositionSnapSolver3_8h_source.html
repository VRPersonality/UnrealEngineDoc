17/**
18 * FBasePositionSnapSolver3 is a base class for 3D position snapping implementations.
19 * It is not usable on its own and the split between this class and the
20 * implementations is not incredibly clean. However there is lots of
21 * shared functionality that is placed here.
22 *
23 * 3D Point and Line targets are supported. Targets can also be "ignored".
24 *
25 * The actual snap "solve" must be implemented by subclasses, depending on
26 * their input data (ray, point, line, etc)
27 *
28 *
29 */
