124 /**
125 * With the introduction of "reroute" (UK2Node_Knot) nodes, we have to
126 * backtrace to find a node that was actually executed before the one in
127 * question (knot nodes are removed at compile time, and therefore never
128 * executed).
129 *
130 * This function recursively searches for any pins feeding the OutputPin (or
131 * the OutputPin itself); it looks for a pin on a non-reroute node (one that
132 * is listed in NodeExecutionList).
133 *
134 * @param OutputPin An exec pin, leading into the node in question (an output pin on some other node).
135 * @param NodeExecutionList A list of pins that feed the execution of a specific node.
136 * @return Null if OutputPin did not cause the invocation of the node in question, otherwise a valid time pair denoting when the node was executed.
