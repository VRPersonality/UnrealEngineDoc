107/**
108 * Intended to be wrapped and used by FBlueprintActionMenuItem. Rather than
109 * sub-classing the menu item, we choose to subclass this instead (for
110 * different node types). That way, we get the type inference that comes with
111 * UObjects (and we don't have to continuously compare identification strings).

198 /**
199 * Takes care of spawning a node for the specified graph. Looks to see if
200 * the supplied graph is transient, and if so, spawns a NOT fully formed
201 * node (intended for template use).
202 *
203 * This function is intended to be overridden; sub-classes may return

208 * @param ParentGraph The graph you want the node spawned into.
