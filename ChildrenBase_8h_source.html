19/**
20 * FChildren is an interface that must be implemented by all child containers.
21 * It allows iteration over a list of any Widget's children regardless of how
22 * the underlying Widget happens to store its children.
23 *
24 * FChildren is intended to be returned by the GetChildren() method.
25 */
26class FChildren

28public:
