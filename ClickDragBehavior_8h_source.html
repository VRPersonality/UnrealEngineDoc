21 * UClickDragInputBehavior implements a standard "button-click-drag"-style input behavior.
22 * An IClickDragBehaviorTarget instance must be provided which is manipulated by this behavior.
23 *
24 * The state machine works as follows:
25 * 1) on input-device-button-press, call Target::CanBeginClickDragSequence to determine if capture should begin
26 * 2) on input-device-move, call Target::OnClickDrag
27 * 3) on input-device-button-release, call Target::OnClickRelease
