10/**
11 * A DamageType is intended to define and describe a particular form of damage and to provide an avenue
12 * for customizing responses to damage from various sources.
13 *
14 * For example, a game could make a DamageType_Fire set it up to ignite the damaged actor.
15 *
16 * DamageTypes are never instanced and should be treated as immutable data holders with static code
17 * functionality. They should never be stateful.
18 */
19UCLASS(MinimalAPI, const, Blueprintable, BlueprintType)
