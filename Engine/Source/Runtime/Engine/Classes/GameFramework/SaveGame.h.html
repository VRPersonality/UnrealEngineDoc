14/**
15 * This class acts as a base class for a save game object that can be used to save state about the game.
16 * When you create your own save game subclass, you would add member variables for the information that you want to save.
17 * Then when you want to save a game, create an instance of this object using CreateSaveGameObject, fill in the data, and use SaveGameToSlot, providing a slot name.
18 * To load the game you then just use LoadGameFromSlot, and read the data from the resulting object.
19 *
20 * @see https://docs.unrealengine.com/latest/INT/Gameplay/SaveGame
21 */

27 * @see UGameplayStatics::SaveGameToSlot
28 * @see UGameplayStatics::DoesSaveGameExist
29 * @see UGameplayStatics::LoadGameFromSlot
30 * @see UGameplayStatics::DeleteGameInSlot
31 */
32
33 GENERATED_BODY()
34};
