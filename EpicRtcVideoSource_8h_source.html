12 /**
13 * A source of video frames for an EpicRtc peer. Has a video input which will provide
14 * frame data to this source. The source will then pass that data to an adapter
15 * which will have one or many adapt processes that are provided by the input object
16 * and are responsible for converting the frame data to the format required for
17 * the selected video encoder.
18 * This video source should be contained within a FVideoSourceGroup which is responsible
19 * for telling each source to push a frame to EpicRtc at the expected rate. This source
20 * will make sure that the adapter has valid output and if so will create a frame
21 * for EpicRtc. Otherwise it will continue to wait until the next frame.
22 */
