30/**
31 * Note: FExtrudeMesh might someday be removed. Consider using FOffsetMeshRegion instead,
32 * which can extrude all or part of a mesh.
33 *
34 * FExtrudeMesh implements a full-mesh extrusion of a mesh. This happens in two stages:
35 * 1) all triangles of input mesh are duplicated and offset
36 * 2) base and offset border loops are stitched together with triangulated quads
37 * Step 2 does not occur if there are no boundary loops (ie for a closed input mesh)
38 *
39 * Each quad of the border loop is assigned it's own normal and UVs (ie each is a separate UV-island)
40 */
