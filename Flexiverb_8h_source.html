37 /********************************************************************************/
38 /* FFlexiverb */
39 /* Flexiverb is a computationally inexpensive single-channel reverb algorithm */
40 /* optimized for maximizing echo density with as few multiplies as possible. */
41 /* This effect is best suited for emulating small room reverberation on */
42 /* platforms where CPU resources are limited. */
43 /* At longer decay times, this reverb algorithm begins to sound very metallic. */
44 /* This algorithm uses a Householder matrix as the scattering matrix for an FDN.*/
45 /* More information on this can be found here: */
46 /* https://ccrma.stanford.edu/~jos/pasp/Householder_Feedback_Matrix.html */
47 /********************************************************************************/
