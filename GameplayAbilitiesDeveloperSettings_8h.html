UCLASS ( )

Expose Global Gameplay Ability Settings in an easy to understand Developer Settings interface (usable through the Editor's Project Settings).

This the preferred way to configure the config variables previously found in AbilitySystemGlobals. Projects may still opt to override the AbilitySystemGlobals class with their own customized class to modify Gameplay Ability functionality across their project.
