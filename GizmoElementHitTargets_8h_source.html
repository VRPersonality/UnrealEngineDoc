60/**

63 * HitTargets which support hitting multiple parts within a gizmo element hierarchy.
64 *
65 * For a gizmo with multiple parts, the part identifier establishes a correspondence between a gizmo part
66 * and the elements representing that part within the hit target. The valid part identifiers should
67 * be defined in the gizmo. Identifier 0 is reserved for the default ID which should be assigned to
68 * elements that do not correspond to any gizmo part, such as non-hittable decorative elements.
