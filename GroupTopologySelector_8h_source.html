144 /**
145 * Using the edges in the given selection as starting points, add any "edge loops" containing the edges. An
146 * edge loop is a sequence of edges that passes through valence-4 corners through the opposite edge, and may
147 * not actually form a complete loop if they hit a non-valence-4 corner.
148 *
149 * @param Selection Selection to expand.
150 * @returns true if selection was modified (i.e., were the already selected edges part of any edge loops whose
151 * member edges were not yet all selected).
152 */
153 UE_API bool ExpandSelectionByEdgeLoops(FGroupTopologySelection& Selection);
154
155 /**
156 * Using the edges in the given selection as starting points, add any "boundary loops" containing the edges. A
157 * boundary loop is just the sequence of edges that surrounds a hole in the mesh, or the outside of an open mesh.
158 * The loop may not complete if it hits a bowtie vertex (a vertex with more than two incident boundary edges).
159 *
160 * @param Selection Selection to expand.
161 * @returns true if selection was modified (i.e., were the already selected edges part of any boundary loops whose
162 * member edges were not yet all selected).
163 */
164 UE_API bool ExpandSelectionByBoundaryLoops(FGroupTopologySelection& Selection);
165
166 /**
167 * Using the edges in the given selection as starting points, add any "edge rings" containing the edges. An
168 * edge ring is a sequence of edges that lie on opposite sides of quad-like faces, meaning faces that have four
169 * corners.
170 *
171 * @param Selection Selection to expand.
172 * @returns true if selection was modified (i.e., were the already selected edges part of any edge rings whose
173 * member edges were not yet all selected).
174 */
175 UE_API bool ExpandSelectionByEdgeRings(FGroupTopologySelection& Selection);
