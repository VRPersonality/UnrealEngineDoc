UINTERFACE ( )

The Controller is the sole authority to perform changes on the Animation Data Model.

Any mutation to the model made will cause a subsequent notify (EAnimDataModelNotifyType) to be broadcast-ed from the Model's ModifiedEvent. Alongside of it is a payload containing information relevant to the mutation. These notifies should be relied upon to update any dependent views or generated (derived) data.
