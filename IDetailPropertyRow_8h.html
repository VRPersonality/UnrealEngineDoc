enum class ECustomEditConditionMode : uint8
strong
Enumerator
Override 

The custom edit condition overrides the edit condition set directly on the property (if any).

Only the custom edit condition must evaluate to true for the property to be editable.

Merge 

The custom edit condition merges with the existing edit condition set directly on the property (if any).

Both edit conditions must evaluate to true for the property to be editable.
