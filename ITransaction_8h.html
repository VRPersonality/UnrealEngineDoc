◆ ETransactionStateEventType

enum class ETransactionStateEventType : uint8
strong

Different kinds of actions that can trigger a transaction state change.

Enumerator
TransactionStarted 

A transaction has been started.

This will be followed by a TransactionCanceled or TransactionFinalized event.

TransactionCanceled 

A transaction was canceled.

PreTransactionFinalized 

A transaction is about to be finalized.

It is still safe to temporarily open new transactions within the scope of this event.

TransactionFinalized 

A transaction was finalized.

PostTransactionFinalized 

A transaction has been finalized and the internal transaction state has been updated.

UndoRedoStarted 

A transaction will be used used in an undo/redo operation.

This will be followed by a UndoRedoFinalized event.

UndoRedoFinalized 

A transaction has been used in an undo/redo operation.
