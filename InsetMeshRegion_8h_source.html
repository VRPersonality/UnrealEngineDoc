23/**
24 * FInsetMeshRegion implements local inset of a mesh region.
25 * The selected triangles are separated and then stitched back together, creating
26 * an new strip of triangles around their border (s). The boundary loop vertices
27 * are inset by creating an offset line for each boundary loop edge, and then
28 * finding closest-points between the sequential edge pairs.
29 *
30 * Complex input regions are handled, eg it can be multiple disconnected components, donut-shaped, etc
31 *
32 * Each quad of the border loop is assigned it's own normal and UVs (ie each is a separate UV-island)
33 */
