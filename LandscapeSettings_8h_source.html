14enum class ELandscapeDirtyingMode : uint8
15{
16 /** Auto :
17 Landscapes that are marked as needing to be resaved will appear in the Choose files to save dialog.
18 Changes are saved whenever the Landscape requires it.*/
19 Auto,
20 /** In Landscape Mode Only :
21 Landscapes that are marked as needing to be resaved will not appear in the Choose files to save dialog.
22 This is a manual saving mode that puts the responsibility on the user to avoid file contention with other team members.
23 The viewport will display an error message indicating that landscape actors are not up-to-date and need to be resaved. This is done using Build > Save Modified Landscapes (or Build > Build Landscape). */
24 InLandscapeModeOnly,
25 /** In Landscape Mode And User Triggered Changes :
26 Landscapes that are marked as needing to be resaved will not appear in the Choose files to save dialog.
27 However, any user-triggered changes (direct or indirect) will require the Landscape to be resaved.
28 This mode is recommended for team collaboration as it provides the best features of the other two modes while ensuring that modified landscape actors are still saved and properly submitted to source control. */
29 InLandscapeModeAndUserTriggeredChanges
30};
