◆ LLF_SerializeCompressed

#define LLF_SerializeCompressed   0x00000002

If set, payload should be [un]compressed during serialization.

Only bulk data that doesn't require any special serialization or endian conversion can be compressed! The code will simply serialize a block from disk and use the byte order agnostic Serialize( Data, Length ) function to fill the memory.
