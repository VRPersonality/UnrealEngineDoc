12/**
13
14 FMallocLeakReporter is a helper class that works alongside FMallocLeakDetection to provide easy access to
15 reports of low-level memory leaks.
16
17 The reporter can be operated from the console where it can be started/stopped and used to periodically
18 or manually report leaks, or it can be called from game code to implement this functionality at desired times.
19
20 Example Console Usage:
21
22 "mallocleak.start report=300" - start tracking leaks and generate a report every 300 secs
23 "mallocleak report" - report allocations and leaks
24 "mallocleak stop" - stop tracking leaks
25
26 Example Code usage
27
28 FMallocLeakReporter::Get().Start(0, 300) - start tracking allocs > 0 bytes and generate a report every 300 secs
29 FMallocLeakReporter::Get().WriteReports() - Writes reports and returns the number of suspected leaks
30 FMallocLeakReporter::Get().WriteReport(TEXT("BigAllocs"), Options); - Write custom report of allocations that matches "Options"
31 FMallocLeakReporter::Get().Stop() - stop tracking allocations
32
33
34 Reports are written to the profiling dir, e,g GameName/Saved/Profiling/SessionName
35
36 */
