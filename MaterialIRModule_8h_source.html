9// This class represents the intermediate representation (IR) of a material build.
10// The IRModule includes an IR value graph, produced through expression analysis,
11// as well as metadata on resource usage and reflection. The IR graph serves as an
12// abstract representation of the material and must be translated into a target backend
13// such as HLSL or specific Preshader opcodes for execution.
14//
15// This class is designed to be backend-agnostic, meaning it does not contain any
16// HLSL code nor does it configure a MaterialCompilationOutput instance. The data
17// stored within this class should be sufficient to enable translation to any supported
18// backend without requiring additional processing or validation.
19class FMaterialIRModule
