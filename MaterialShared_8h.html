◆ ENUM_CLASS_FLAGS()

ENUM_CLASS_FLAGS ( EMaterialTranslateValidationFlags )

Enum that contains entries for the ways that material properties need to be compiled.

This 'inherits' from EMaterialProperty in the sense that all of its values start after the values in EMaterialProperty. Each material property is compiled once for its usual shader frequency, determined by GetShaderFrequency(), And then this enum contains entries for extra compiles of a material property with a different shader frequency. This is necessary for material properties which need to be evaluated in multiple shader frequencies.
