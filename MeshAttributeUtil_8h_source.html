22 /**
23 * Compact the values of an integer Triangle Attribute, ie so that the attribute values are dense in range 0..N.
24 * Useful for (eg) compacting MaterialIDs or Polygroups.
25 * Note that this function cannot tell if values were removed from the "end" of the attribute value list. Some care must
26 * be taken in calling code to handle this case if using this function to determine whether compaction of other data structures is needed.
27 * Currently OldToNewMapOut and NewToOldMapOut are always populated even if the input is compact.
28 * @param OldMaxAttributeRangeOut min and max input attribute values, ie pre-compaction
29 * @param NewMaxAttributeValueOut new max attribute value after compacting.
30 * @param OldToNewMapOut generated mapping from previous IDs to new IDs (ie size is maximum input attrib value (+1), will contain InvalidID for any compacted values )
31 * @param NewToOldMapOut generated mapping from new IDs to previous IDs (ie size is maximum output attribute value (+1), no invalid values)
32 * @param bWasCompactOut set to true if the attribute set was already compact, IE was not modified by the operation
33 * @return true on success (including if already compact). false if any Attribute Values were negative/InvalidID, cannot compact in this case
34 */
35 GEOMETRYCORE_API bool CompactAttributeValues(
36 const FDynamicMesh3& Mesh,
37 TDynamicMeshScalarTriangleAttribute<int32>& TriangleAttrib,
38 FInterval1i& OldMaxAttributeRangeOut,
39 int& NewMaxAttributeValueOut,
40 TArray<int32>& OldToNewMapOut,
41 TArray<int32>& NewToOldMapOut,
42 bool& bWasCompactOut);
