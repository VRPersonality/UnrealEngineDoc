193/**
194* Class that, given a starting location and direction, traces a geodesic path along a mesh.
195* As the path enters a triangle, it is unfolded a two-dimensional triangle and the path is transformed into
196* the local frame of the triangle.
197*
198* At each point where the path crosses into a new triangle, this class records the point and direction
199* in terms of barycentric coordinates, triangle references and local path direction relative
200* to a local frame (in the form of one of the edges of the triangle).
201*/

359/**
360* return AngleR in [0,2pi) range.
361*
362* The same polar point (r, AngleR) can be described by an infinite number of alternative
363* angles, each offset by some multiple of 2pi, i.e. (r, AngleR + n * 2Pi).
364* This selects the unique angle that falls in the [0,2pi) range.
365*/
366static double AsZeroToTwoPi(double AngleR)
367{
368 while (AngleR < 0.)
369 {
370 AngleR += TMathUtilConstants<double>::TwoPi;
371 }
372
373 while (AngleR > TMathUtilConstants<double>::TwoPi)
374 {
375 AngleR -= TMathUtilConstants<double>::TwoPi;
376 }
377 return AngleR;
378}
