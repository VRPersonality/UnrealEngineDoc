307 /**
308 * Update the Normals and/or Tangents at VertexID of TargetMesh. Note that the specified vertex may have "split normals"
309 * or "split tangents", ie in the case of hard/crease normals, UV seams, and so on. In these situations, by default
310 * each of the unique normals/tangents at the vertex will be updated, but they will not be "merged", ie they will remain split.
311 * However if bMergeSplitValues=true, then the vertex will be "un-split", ie after the function call the vertex will have
312 * a single unique shared normal and/or tangents.
313 *
314 * Note that this function requires that some normals/tangents already exist on the TargetMesh. If this is not the case,
315 * functions like SetPerVertexNormals and ComputeTangents can be used to initialize the normals/tangents first.
316 *
317 * @param bUpdateNormal if true (default) then the normals overlay is updated
318 * @param NewNormal the new normal vector. This vector will not be normalized, it must be normalized by the calling code.
319 * @param bUpdateTangents if true then the tangents overlay will be updated. If the tangents overlay does not exist, this function returns an error.
320 * @param NewTangentX the new tangent vector. This vector will not be normalized, it must be normalized by the calling code.
321 * @param NewTangentY the new bitangent/binormal vector. This vector will not be normalized, it must be normalized by the calling code.
322 * @param bIsValidVertex will be set to true on return if the VertexID was valid, ie had valid normals and tangents
323 * @param bMergeSplitValues if true, any split normals/tangents at the vertex will be cleared, and a unique normal/tangent element will be set in the connected triangles
324 * @param bDeferChangeNotifications if true, no mesh change notification will be sent. Set to true if changing many normals in a loop.
325 */
326 UFUNCTION(BlueprintCallable, Category = "GeometryScript|MeshEdits", meta=(ScriptMethod, bUpdateNormal="true"))
327 static UE_API UPARAM(DisplayName = "Target Mesh") UDynamicMesh*
328 UpdateVertexNormal(
329 UDynamicMesh* TargetMesh,
330 int VertexID,
331 bool bUpdateNormal,
332 FVector NewNormal,
333 bool bUpdateTangents,
334 FVector NewTangentX,
335 FVector NewTangentY,
