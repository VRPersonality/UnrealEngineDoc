77 /**
78 * Assumes that ComputeTriangleNormals() has already been run.
79 *
80 * Looks through the normals for any that are zero, and tries to set them to the normal of
81 * some neighboring triangle, preferring the triangle on its longer side if possible. This is
82 * useful for finding connected components based on normals, as it prevents degenerate triangles
83 * with 0 normals from connecting otherwise disconnected components.
84 * This won't help if a whole mesh componenent is entirely made of such degenerate triangles
85 * (which is usually ok).
86 */
87 GEOMETRYCORE_API void SetDegenerateTriangleNormalsToNeighborNormal();
