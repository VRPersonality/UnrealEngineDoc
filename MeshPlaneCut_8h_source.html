33/**
34 * Cut the Mesh with the Plane. The *positive* side, ie (p-o).n > 0, is removed.
35 * If possible, returns boundary loop(s) along cut
36 * (this will fail if cut intersected with holes in mesh).
37 * Also FillHoles() for a topological fill. Or use CutLoops and fill yourself.
38 *
39 * Algorithm is:
40 * 1) find all edge crossings
41 * 2) optionally discard any triangles with all vertex distances < epsilon.
42 * 3) Do edge splits at crossings
43 * 4 option a) (optionally) delete all vertices on positive side
44 * 4 option b) (OR optionally) disconnect all triangles w/ vertices on positive side (if keeping both sides)
45 * 4 option c) do nothing (if keeping both sides and not disconnecting them)
46 * 5) (optionally) collapse any degenerate boundary edges
47 * 6) (optionally) change an attribute tag for all triangles on positive side
48 * 7) find loops through valid boundary edges (ie connected to splits, or on-plane edges) (if second half was kept, do this separately for each separate mesh ID label)
49 */
