18/**
19 * A utility class that lets you store a collection of timestamped data originating from various time bases.
20 *
21 * All of the data is stored in "root" time space. That is: as you add timestamped data, these timestamps are
22 * converted back to "root times" using the inverse of the current time transform. Pushing and popping time
23 * transforms, and incrementing loop counts, makes it possible to change what's considered "local time",
24 * which affects this inverse transformation.
25 */
