16// UNetworkPredictionComponent
17// This is the base component for running a TNetworkedSimulationModel through an actor component. This contains the boiler plate hooks into getting the system
18// initialized and plugged into the UE replication system.
19//
20// This is an abstract component and cannot function on its own. It must be subclassed and InitializeNetworkPredictionProxy must be implemented.
21// Ticking and RPC sending will be handled automatically.
22//
23// Its also worth pointing out that nothing about being a UActorComponent is essential here. All that this component does could be done within an AActor itself.
24// An actor component makes sense for flexible/reusable code provided by the plugin. But there is nothing stopping you from copying this directly into an actor if you had reason to.
25// ----------------------------------------------------------------------------------------------------------------------------------------------------------------------
