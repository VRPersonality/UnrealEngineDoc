14/**
15 * Remeshing with "face aligned projection". This approach to projection attempts to preserve sharp features in the
16 * mesh by aligning triangle normals with normals from the original mesh during the projection pass.
17 * This class also performs a pass of edge flips aimed at further aligning triangle normals to their originals. This
18 * can be helpful in fixing the occasional single "bad edge" along a sequence of feature edges.
19 */

127 /**
128 Test to see if flipping an edge would improve "normal error". Normal error for a triangle is:
129 0.5 * (1.0 - TriangleNormal \cdot ProjectedNormal),
130 where ProjectedNormal is the normal of the triangle returned by finding the closest point on ProjTarget. The normal
131 error for an edge is the sum of errors for the edge's incident triangles.
132
133 @param EdgeID Edge to consider.
134 @param BadEdgeErrorThreshold Only edges with this error or greater are candidates for flipping.
135 @param ImprovementRatioThreshold New edge must have error less than old edge error times this ratio.
136
137 @return True iff (CurrentEdgeError > BadEdgeErrorThreshold) and (NewEdgeError < ImprovementRatioThreshold * CurrentEdgeError)
138 */
139 UE_API bool EdgeFlipWouldReduceNormalError(int EdgeID, double BadEdgeErrorThreshold = 0.01, double ImprovementRatioThreshold = 0.75) const;
140
141 /** Flip all edges for which EdgeFlipWouldReduceNormalError returns true */
