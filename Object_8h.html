◆ EEditChangePropagationFlags

enum class EEditChangePropagationFlags : uint8
strong

Flags to control how editor change events get propagated.

Enumerator
None 

No flags; use default change event propagation behavior.

OnlyMarkRealignedInstancesAsDirty 

If set, only the subset of affected instances that realign with their archetype will be marked dirty.

If not set, affected instances will always be marked dirty, even if they already match the archetype.
