UENUM() enum class EPathedPhysicsPlaybackBehavior USTRUCT ( )

Properties that can change during game-time and affect path movement calculations.

Split into its own struct because both the input and sync state structs want to hold onto these.

The server frame of the simulation when movement should/did actually begin Tracked this way so that we can set a future frame as the actual movement start frame to account for network latency, allowing clients and server to start at the same time irl. A value < 0 indicates that no movement is happening along the path.
