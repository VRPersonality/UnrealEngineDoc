21 * This includes various types of indices that could be interpreted in different ways.
22 *
23 * In addition, "Render" geometry is stored, which can be used by higher-level
24 * code to draw the selection in some way (eg a selection highlight)
25 *
26 * @warning this class will be removed in the future
