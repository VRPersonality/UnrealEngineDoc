49// There is a lot of overlap in the options for Extrude, Offset, and Push/Pull, and they map to
50// the same op behind the scenes. However, we want to keep them as separate buttons to keep some
51// amount of shallowness in the UI, to make it more likely that new users will find the setting
52// they are looking for.
53// A couple of settings are entirely replicated: namely, doing an offset or "extrude" with SelectedTriangleNormals
54// or SelectedTriangleNormalsEven as the movement direction is actually equivalent. Properly speaking, these
55// two should only be options under Offset, not Extrude, but we keep them as (non-default) options
56// in both because an "extrude along local normals" is a common operation that some users are likely
57// to look for under extrude, regardless of it not lining up with the physical meaning of extrusion.
58
59// We use different property set objects so that we can customize category names, etc, as well as
60// have different defaults and saved settings.
61
