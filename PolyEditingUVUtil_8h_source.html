32/**
33 * Compute the total UVLength/MeshLength ratio along the given VertexPath.
34 * Other 3D lengths can then be scaled to comparable UV-space lengths by multiplying by this value.
35 * Edges must exist between the vertices along the path, and loops are not considered.
36 * Each UV edge for a 3D edge is considered, ie if a 3D edge corresponds to 2 UV edges, both are counted,
37 * and if the 3D edge has no UV edges, it is skipped.
38 * @param VertexPath list of vertices which are connected by edges
39 * @param PathLengthOut optional output param for sum of 3D edge lengths along path (arc length)
40 * @param UVPathLengthOut optional output param for sum of UV edge lengths along path
41 * @return UVPathLength / PathLength, or 1.0 if no valid UV edges along the path existed
42 */
