15/**
16 * This class facilitates the movement of a dragged edge and recalculates the key heights for all specified keys.
17 * The heights are linearly interpolated, ensuring that if the dragged edge crosses to the opposite side of the mirror edge,
18 * the keys will flip to the other side.
19 *
20 * Inputs:
21 * - Height of the dragged edge aligned to the x-axis
22 * - Height of the x-axis-aligned mirror edge
23 * - Keys originally located between the dragged and mirror edges
24 *
25 * Example:
26 * Given a grid with a single key at a height 100, moving the top edge:
27 * - An equal distance past the bottom edge will result in a height of -100 for the key.
28 * - Halfway between the starting position and the bottom edge will result in a height of 50 for the key.
29 *
30 * @note This class only recalculates key positions. To adjust the tangents of the keys use e.g. FTangentMirrorSolver.
31 */
