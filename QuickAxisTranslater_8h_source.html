17/**
18 * FQuickAxisTranslater implements the underpinnings for "quick" axis transformations, ie axis-gizmo-like
19 * behavior without having to explicitly click on an axis.
20 *
21 * To use this class, you first configure the internal world-axis-frame using the SetActiveX() functions.
22 * Then as you collect input updates, you call UpdateSnap() with the input ray, and this will output
23 * a snapped 3D world-space point. The delta (Frame.Origin - SnapPoint) is the move axis.
24 *
25 * You must also call UpdateCameraState() each time the camera changes (typically each frame in a tool Render())
26 *
27 * A default visualization is provided via the Render() function
28 *
29 * A small snap-ball around the frame origin prevents small movements, which are unstable with this approach.
30 * @todo add ability to do small movements once an axis is chosen?
31 */
