66 // We want the revolution degrees to be clamped to 360 when there is no offset along axis, and extendable
67 // beyond 360 when there is one (to make springs, etc). There's not currently a way to have a conditional
68 // clamp value, so instead, we swap the variables that we make visible to the user (RevolveDegrees and
69 // RevolveDegreesClamped). However, we have to do some work to keep the values consistent so that they
70 // do not jump when they are swapped.
71 ClampedRevolutionDegreesWatcherIndex = WatchProperty(RevolveDegreesClamped,
72 [this](const double& NewClamped)
73 {
74 RevolveDegrees = NewClamped;
