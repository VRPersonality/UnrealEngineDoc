10/**
11* A BackgroundBlur is similar to a Border in that it can be used to contain other widgets.
12* However, instead of surrounding the content with an image, it applies a post-process blur to
13* everything (both actors and widgets) that is underneath it.
14*
15* Note: For low-spec machines where the blur effect is too computationally expensive,
16* a user-specified fallback image is used instead (effectively turning this into a Border)
17*/
