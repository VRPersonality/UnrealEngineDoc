21/**
22 * Implements an overlay widget.
23 *
24 * Overlay widgets allow for layering several widgets on top of each other.
25 * Each slot of an overlay represents a layer that can contain one widget.
26 * The slots will be rendered on top of each other in the order they are declared in code.
27 *
28 * Usage:
29 * SNew(SOverlay)
30 * + SOverlay::Slot(SNew(SMyWidget1))
31 * + SOverlay::Slot(SNew(SMyWidget2))
32 * + SOverlay::Slot(SNew(SMyWidget3))
33 *
34 * Note that SWidget3 will be drawn on top of SWidget2 and SWidget1.
35 */
