16/**
17 * A ScissorRectBox is a widget that clips all its children using a scissor rect that matches it clipping rect
18 * in LAYOUT SPACE (the way it is provided in OnPaint).
19 * This ensures that any child widgets with render transforms do not "bleed out" of this parent container.
20 * Note there is a significant performance impact to using one of these widgets because any widgets inside
21 * this one end up in a separate set of batches using that scissor rect.
22 *
23 * Keep in mind that this widget will use the LAYOUT clipping rect to clip children, since hardware clipping
24 * rects do not support non-aligned rectangles. If a render transform is applied to this widget itself, the results
25 * will not be as expected.
26 *
27 * NOTE: This class is provided as a STOPGAP for design-time widgets (like the UMG designer itself)
28 * until a more robust clipping solution can be put in place.
29 */
