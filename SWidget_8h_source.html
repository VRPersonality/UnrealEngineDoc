130/**
131 * Abstract base class for Slate widgets.
132 *
133 * STOP. DO NOT INHERIT DIRECTLY FROM WIDGET!
134 *
135 * Inheritance:
136 * Widget is not meant to be directly inherited. Instead consider inheriting from LeafWidget or Panel,
137 * which represent intended use cases and provide a succinct set of methods which to override.
138 *
139 * SWidget is the base class for all interactive Slate entities. SWidget's public interface describes
140 * everything that a Widget can do and is fairly complex as a result.
141 *
142 * Events:
143 * Events in Slate are implemented as virtual functions that the Slate system will call
144 * on a Widget in order to notify the Widget about an important occurrence (e.g. a key press)
145 * or querying the Widget regarding some information (e.g. what mouse cursor should be displayed).
146 *
147 * Widget provides a default implementation for most events; the default implementation does nothing
148 * and does not handle the event.
149 *
150 * Some events are able to reply to the system by returning an FReply, FCursorReply, or similar
151 * object.
152 */

1637 /**
1638 * Registers an "active timer" delegate that will execute at some regular interval. TickFunction will not be called until the specified interval has elapsed once.
1639 * A widget can register as many delegates as it needs. Be careful when registering to avoid duplicate active timers.
1640 *
1641 * An active timer can be UnRegistered in one of three ways:

1644 * 3. Destroying the widget
1645 *
1646 * Active Timers
1647 * --------------
1648 * Slate may go to sleep when there is no user interaction for some time to save power.
1649 * However, some UI elements may need to "drive" the UI even when the user is not providing any input
1650 * (ie, animations, viewport rendering, async polling, etc). A widget notifies Slate of this by
1651 * registering an "Active Timer" that is executed at a specified frequency to drive the UI.

1653 * When any active timer needs to fire, all of Slate will do a Tick and Paint pass.
1654 *
1655 * @param Period The time period to wait between each execution of the timer. Pass zero to fire the timer once per frame.
1656 * If an interval is missed, the delegate is NOT called more than once.
