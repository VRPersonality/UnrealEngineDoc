◆ ENABLE_VECTORIZED_TRANSFORM

#define ENABLE_VECTORIZED_TRANSFORM   PLATFORM_ENABLE_VECTORINTRINSICS

This define controls whether a scalar implementation or vector implementation is used for FTransform.

The vector implementation works even when using UnMathFPU, but it will be much slower than the equivalent scalar implementation, so the scalar code is maintained and enabled when vector intrinsics are off.
