21/**
22* The ScalarRegister class wraps the concept of a 'float-in-vector', allowing common scalar operations like bone
23* weight calculations to be done in vector registers. This will avoid some LHS hazards that arise when mixing float
24* and vector math on some platforms. However, doing the math for four elements is slower if the vector operations are
25* being emulated on a scalar FPU, so ScalarRegister is defined to float when ENABLE_VECTORIZED_TRANSFORM == 0.
26*/
