enum class EBlueprintTextLiteralType : uint8
strong
Enumerator
Empty 
LocalizedText 

Text is localized.

The bytecode will contain three strings - source, key, and namespace - and should be loaded via FInternationalization

InvariantText 

Text is culture invariant.

The bytecode will contain one string, and you should use FText::AsCultureInvariant to initialize the FText instance.

LiteralString 

Text is a literal FString.

The bytecode will contain one string, and you should use FText::FromString to initialize the FText instance.

StringTableEntry 

Text is from a string table.

The bytecode will contain an object pointer (not used) and two strings - the table ID, and key - and should be found via FText::FromStringTable
