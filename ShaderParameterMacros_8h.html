Implements the same contract as IMPLEMENT_STATIC_UNIFORM_BUFFER_STRUCT, with the addition of the 'StaticAndShader' binding flag.

This means that the uniform buffer may be bound statically or through SetShaderParameters. The uniform buffer is NOT removed from the parameter map, which means it is possible to end up with both global and shader bindings. The RHI validation layer will catch and report these cases, as they are both inefficient and would produce confusing behavior if the two bindings differ. Only one binding model should be used at a time.
