35/**
36 * USingleClickTool is perhaps the simplest possible interactive tool. It simply
37 * reacts to default primary button clicks for the active device (eg left-mouse clicks).
38 *
39 * The function ::IsHitByClick() determines what is clickable by this Tool. The default is
40 * to return true, which means the click will activate anywhere (the Tool itself has no
41 * notion of Actors, Components, etc). You can override this function to, for example,
42 * filter out clicks that don't hit a target object, etc.
43 *
44 * The function ::OnClicked() implements the action that will occur when a click happens.
45 * You must override this to implement any kind of useful behavior.
