30/**
31 * Smooth scalar data of any dimension that is stored via TDynamicMeshOverlay. The operator doesn't know anything about
32 * constraints that are associated with the actual overlay type. For example, if the overlay contains unit length
33 * normals, the resulting "smoothed" normals might not have unit length.
34 *
35 * The smoothing happens in stages:
36 *
37 * - First, we average the overlay element values at each vertex to obtain the per-vertex values.
38 *
39 * - For each vertex, we compute the weighted average of its connected neighbors using the EdgeWeightMethod weights.
40 * The smoothed value equals the (1-Strength)*PerVertexValue + (Strength)*NeighborAverage. Strength is expected to be
41 * in the zero to one range, otherwise the operation can turn from smoothing to more of an unsharp filter.
42 * We repeat this NumIterations times.
43 *
44 * - Finally, the smoothed per-vertex values are written back to the overlay by either overwriting the content of
45 * all per-vertex elements (SplitVertexModel == ESplitVertexModel::SetAllToSmoothed) or blending on per-element
46 * basis using the BlendWithSmoothedStrength blend value. The calling code could use FSplitAttributeWelder to weld
47 * split-attributes that happen to have identical values after this smoothing (happens when the
48 * SplitVertexModel == ESplitVertexModel::SetAllToSmoothed).
49 *
50 *
51 * Example Usage:
52 *
53 * // Smoothing the color
54 * FDynamicMesh3 Mesh = ...
55 * FDynamicMeshColorOverlay* Colors = Mesh.Attributes()->PrimaryColors();
56 *
57 * FSmoothDynamicMeshAttributes BlurOp(&Mesh);
58 *
59 * // Setup the parameters
60 * BlurOp.bUseParallel = true;
61 * BlurOp.NumIterations = 10;
62 * BlurOp.Strength = 0.2;
63 * BlurOp.EdgeWeightMethod = EEdgeWeights::CotanWeights;
64 *
65 * // Run the smoothing
66 * BlurOp.SmoothOverlay(Colors);
67 */
