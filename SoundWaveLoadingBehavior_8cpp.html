◆ EnumToString()

const TCHAR * EnumToString ( ESoundWaveLoadingBehavior InCurrentState)

Only used when stream caching is enabled.

Determines how we are going to load or retain a given audio asset. A USoundWave's loading behavior can be overridden in the USoundWave itself, the sound wave's USoundClass, or by cvars. The order of priority is defined as: 1) The loading behavior set on the USoundWave 2) The loading behavior set on the USoundWave's USoundClass. 3) The loading behavior set on the nearest parent of a USoundWave's USoundClass. 4) The loading behavior set via the au.streamcache cvars.
