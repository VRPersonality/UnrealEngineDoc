Delegate used by the execution context to collect external data views for a given StateTree asset.

The caller is expected to iterate over the ExternalDataDescs array, find the matching external data, and store it in the OutDataViews at the same index:

for (int32 Index = 0; Index < ExternalDataDescs.Num(); Index++) { const FStateTreeExternalDataDesc& Desc = ExternalDataDescs[Index]; // Find data requested by Desc OutDataViews[Index] = ...; }
