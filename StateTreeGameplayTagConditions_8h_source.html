78/**
79 * HasAnyTags / HasAllTags condition
80 * Succeeds if the specified tag container has ANY or ALL of the tags in the other container.
81 *
82 * Condition can be used with multiple configurations:
83 *
84 * Has ANY Tags:
85 * exact match 'false':
86 * TagContainer {"A.1"} has any of OtherContainer {"A","B"} will SUCCEED
87 * TagContainer {"A"} has any of OtherContainer {"A.1","B"} will FAIL
88 *
89 * exact match 'true':
90* TagContainer {"A","B"} has any of OtherContainer {"A.1"} will FAIL
91 *
92 * If TagContainer is empty/invalid it will always fail.
93 *
94 * Has ALL Tags:
95 * exact match 'false':
96 * TagContainer {"A.1","B.1"} has all of OtherContainer {"A","B"} will SUCCEED
97 * TagContainer {"A","B"} has all of OtherContainer {"A.1","B.1"} will FAIL
98 *
99 * exact match 'true':
100 * TagContainer {"A.1","B.1"} has all of OtherContainer {"A","B"} will FAIL
101 *
102 * If TagContainer is empty/invalid it will always SUCCEED, because there were no failed checks.
