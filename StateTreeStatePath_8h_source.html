110 /**
111 * Describe the state list used to get to a specific state.
112 * Since a state can be a subtree and can be linked to other states, the path to activate the subtree can be different.
113 * A state can enter, then exit, then reenter. It is considered the same state path but not the same unique state path.
114 * For a tree like
115 * RootA(0)
116 * StateA(1)
117 * StateB(2)
118 * LinkedStateA(3)
119 * StateC(4)
120 * LinkedStateA(5)
121 * SubTreeA(6)
