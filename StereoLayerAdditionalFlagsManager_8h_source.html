10/**
11 * To add flags to StereoLayerComponents from plugins, please follow this procedure:
12 * 1. Implement the IStereoLayersFlagsSupplier interface.
13 *
14 * 2. Register the modular feature:
15 * IModularFeatures::Get().RegisterModularFeature(IStereoLayersFlagsSupplier::GetModularFeatureName(), this);
16 *
17 * 3. Unregister the modular feature when the plugin is torn down:
18 * IModularFeatures::Get().UnregisterModularFeature(IStereoLayersFlagsSupplier::GetModularFeatureName(), this);
19 *
20 * 4. Implement the EnumerateFlags(TSet<FName>& OutFlags) function, where you can add your flags. A simple
21 * implementation might be:
22 * void MyFlagSupplier::EnumerateFlags(TSet<FName>& OutFlags)
23 * {
24 * OutFlags.Add(FName("MY_FLAG_1"));
25 * OutFlags.Add(FName("MY_FLAG_2"));
26 * }
27 * 5. It's up to the developers to make sure the flag values are not overflowing.
28 * 6. At runtime, use
29 * TSharedPtr<FStereoLayerAdditionalFlagsManager> FlagsManager = FStereoLayerAdditionalFlagsManager::Get();
30 * FlagsManager->GetFlagValue(Flag);
31 * to get flags value to check against the flags in your StereoLayerComponents.
32 */
