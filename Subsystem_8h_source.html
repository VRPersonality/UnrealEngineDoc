12/** Subsystems are auto instanced classes that share the lifetime of certain engine constructs
13 *
14 * Currently supported Subsystem lifetimes are:
15 * Engine -> inherit UEngineSubsystem
16 * Editor -> inherit UEditorSubsystem
17 * GameInstance -> inherit UGameInstanceSubsystem
18 * World -> inherit UWorldSubsystem
19 * LocalPlayer -> inherit ULocalPlayerSubsystem
20 *
21 *
22 * Normal Example:
23 * class UMySystem : public UGameInstanceSubsystem
24 * Which can be accessed by:
25 * UGameInstance* GameInstance = ...;
26 * UMySystem* MySystem = GameInstance->GetSubsystem<UMySystem>();
27 *
28 * or the following if you need protection from a null GameInstance
29 * UGameInstance* GameInstance = ...;
30 * UMyGameSubsystem* MySubsystem = UGameInstance::GetSubsystem<MyGameSubsystem>(GameInstance);
31 *
32 *
33 * You can get also define interfaces that can have multiple implementations.
34 * Interface Example :
35 * MySystemInterface
36 * With 2 concrete derivative classes:
37 * MyA : public MySystemInterface
38 * MyB : public MySystemInterface
39 *
40 * Which can be accessed by:
41 * UGameInstance* GameInstance = ...;
42 * const TArray<UMyGameSubsystem*>& MySubsystems = GameInstance->GetSubsystemArray<MyGameSubsystem>();
43 *
44 */
45
