17/**
18 * Texture2DBuilder is a utility class for creating/modifying various types of UTexture2D.
19 * Use Initialize() functions to configure, can either generate a new UTexture2D (in the Transient package) or modify an existing UTexture2D.
20 *
21 * Currently the generated UTexture2D will only have Mip 0, and only Mip 0 can be edited.
22 * The generated UTexture2D has format PF_B8G8R8A8 or PF_FloatRGBA.
23 *
24 * Use Commit() to lock and update the texture after editing is complete. LockForEditing() can be used to re-open.
25 * By default textures are locked for editing on Initialize()
26 *
27 * If you have generated a UTexture2D by other means, you can use the static function ::CopyPlatformDataToSourceData() to populate the
28 * Source data from the PlatformData, which is required to save it as a UAsset.
29 */

341 /**
342 * Create a new UTexture2D of the given TextureType from the given SourceImage
343 * @param bConvertToSRGB if true, assumption is SourceImage is Linear and we want output UTexture2D to be SRGB. Only valid for 8-bit textures.
344 * @param bPopulateSourceData if true, first the PlatformData is initialized, then it is copied to the SourceData. This is necessary if the UTexture2D is going to be stored as an Asset (but can also be done explicitly using Commit())
345 */
346 template<typename PixelType>
347 static UTexture2D* BuildTextureFromImage(const TImageBuilder<PixelType>& SourceImage, FTexture2DBuilder::ETextureType TextureType, const bool bConvertToSRGB, bool bPopulateSourceData = true)
348 {
349 FTexture2DBuilder TexBuilder;
350 TexBuilder.Initialize(TextureType, SourceImage.GetDimensions());
351 TexBuilder.Copy(SourceImage, bConvertToSRGB);
352 TexBuilder.Commit(bPopulateSourceData);
353 return TexBuilder.GetTexture2D();
354 }
