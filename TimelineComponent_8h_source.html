205 uint8 bPlaying_DEPRECATED:1;
206
207 /**
208 * If the first bit is set to 1 (PlayingStateTracker & 0x01 == 1), then we are playing
209 *
210 * The rest of the bits in the uint8 are reserved for keeping track of the "dirty" state,
211 * being incremented when our state is modified. This ensures that the value is replicated
212 * if it changes multiple times in one frame, such as calling "Play From Start" in the resulting
213 * "Finished" delegate.
214 *
215 * You should modify this value using the "ChangeMarkPlayingState" function.
216 */
