95 // For performance reasons, we store static traits that never unload into
96 // a single contiguous memory buffer. However, traits cannot be guaranteed
97 // to be trivially copyable because they contain virtual functions. As such,
98 // we cannot resize the buffer once they have been allocated. We reserve a fixed
99 // amount of space that should easily cover our needs. Static traits are
100 // generally stateless and only contain a few v-tables. Their size is usually
101 // less than 32 bytes. Additionally, we will likely only ever load a few hundred
102 // traits. If we exceed the size of the buffer, additional traits will
103 // be treated as dynamic. Dynamic traits are instead allocated on the heap.
104 static constexpr uint32 STATIC_TRAIT_BUFFER_SIZE = 8 * 1024;
