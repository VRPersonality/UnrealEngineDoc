14/**
15 * Helper class for binding a UCombinedTransformGizmo to several FVector3d's such that they
16 * are updated when the gizmo is changed, and the gizmo is updated from the vectors if the
17 * approriate function is called. The vectors can serve as a display of the gizmo's values,
18 * and if they are made modifiable, they can serve as a numerical control of the gizmo.
19 *
20 * To use, just call InitializeBoundVectors with 3 vectors (it can be fewer, but the binder
21 * currently does the same amount of work regardless of whether 3 or fewer are bound), and
22 * then either call BindToGizmoContextObject with the context object that will be used to
23 * create the gizmos (in which case the vectors will bind to all newly created gizmos, and
24 * will switch bindings as different gizmos are dragged), or just bind to a particular gizmo
25 * using BindToInitializedGizmo after its SetActiveTarget call has been made..
26 */
27class FTransformGizmoDataBinder : public TSharedFromThis<FTransformGizmoDataBinder, ESPMode::ThreadSafe>

89 /**
90 * In delta mode, the bound values hold the difference of the current gizmo transform relative

92 * bound values represent the final gizmo transform.

108 /**
109 * Sets to destination mode (as opposed to delta mode). Calling this will update the bound values
110 * to reflect the current tracked gizmo transform. If the gizmo is in local coordinate mode and it
111 * has a custom reference transform, that will be used as the reference.
112 */
113 UE_API virtual void SetToDestinationMode();
114
115 /**
116 * If the gizmo doesn't have all three axes, then it likely needs to stay constrained in some
117 * plane/line. In this situation, allowing destination mode is only safe if the gizmo has a
118 * DisplaySpaceTransform that (presumably) orients the measurement axes correctly, or if the
119 * coordinate mode is global, in which case the world axes are known to be fine.
120 *
121 * Currently, the binder will automatically switch to delta mode when tracking a new gizmo
122 * that doesn't satisfy the above definition of safety, though it won't prevent the mode from
123 * being switched from there...
124 */
125 UE_API virtual bool ShouldDestinationModeBeAllowed();

133 * tracked gizmo being visible.

135 UE_API virtual void SetTrackedGizmo(UCombinedTransformGizmo* Gizmo);
