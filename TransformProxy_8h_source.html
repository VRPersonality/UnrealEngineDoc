26/**
27 * UTransformProxy is used to transform a set of sub-objects. An internal
28 * FTransform is generated based on the sub-object set, and the relative
29 * FTransform of each sub-object is stored. Then as this main transform
30 * is updated, the sub-objects are also updated.
31 *
32 * Currently only USceneComponent sub-objects are supported.
33 *
34 * If only one sub-object is set, the main transform is the sub-object transform.
35 * Otherwise the main transform is centered at the average origin and
36 * has no rotation.
37 */
38UCLASS(Transient, MinimalAPI)
