UDynamicMeshGenerator is an abstract base class for an implementation that can mutate an input mesh into an output mesh.

A subclass of this class can be attached to a UDynamicMesh to allow for arbitrarily-complex procedural generation EDynamicMeshChangeType is used by FDynamicMeshChangeInfo to indicate a "type" of mesh change FDynamicMeshChangeInfo stores information about a change to a UDynamicMesh. This struct is emitted by the UDynamicMesh OnPreMeshChanged() and OnMeshChanged() delegates.
