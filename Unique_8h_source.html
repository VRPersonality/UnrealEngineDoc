39 /**
40 * Eliminates all but the first element from every consecutive group of equivalent elements and
41 * returns past-the-end index of unique elements for the new logical end of the range.
42 *
43 * Removing is done by shifting the elements in the range in such a way that elements to be erased are overwritten.
44 * Relative order of the elements that remain is preserved and the physical size of the range is unchanged.
45 * References to an element between the new logical end and the physical end of the range are still
46 * dereferenceable, but the elements themselves have unspecified values. A call to `Unique` is typically followed by a call
47 * to a container's `SetNum` method as:
48 *
49 * ```Container.SetNum(Algo::Unique(Container));```
50 *
51 * that erases the unspecified values and reduces the physical size of the container
52 * to match its new logical size.
53 *
54 * Elements are compared using operator== or given binary predicate. The behavior is undefined if it is not an equivalence relation.
55 *
56 * See https://en.cppreference.com/w/cpp/algorithm/unique
57 */
