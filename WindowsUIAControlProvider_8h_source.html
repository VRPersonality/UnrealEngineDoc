67/**
68 * The control Provider handles all control pattern-related functionality for the widget Provider. Control Providers should only
69 * be generated through widget Providers in their GetPatternProvider() function. In doing this, a control Pattern is only guaranteed
70 * to support functions that match the pattern Provider that it was generated for. This means that if GetPatternProvider() is called
71 * for UIA_ValuePatternId, the control Provider is only guaranteed to work with functions that implement IValueProvider. It is up
