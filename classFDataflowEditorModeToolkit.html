Detailed Description

The dataflow editor mode toolkit is responsible for the panel on the side in the dataflow editor that shows mode and tool properties.

Tool buttons would go in Init(). Note: When there are separate viewports/worlds/modemanagers/toolscontexts, this ModeToolkit will track which one is currently active.
