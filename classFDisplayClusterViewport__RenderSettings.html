◆ AssignParentViewport()

void FDisplayClusterViewport_RenderSettings::AssignParentViewport ( const FString & InParentViewportId,
const FDisplayClusterViewport_RenderSettings & InParentSettings )
inline

Assign parent viewport to this.

The main idea is copiing some render settings and math from the parent viewport. This is used for a 'link' projection policy to render LC and CK from the same frustum assigned as the parent (outer for LC, incamera for CK). Also, child viewports are only updated when the parent viewport is updated (sorted in ViewportManager/Proxy).

Parameters
InParentViewportId- parent viewport name
InParentSettings- parent viewport rendering settings.
