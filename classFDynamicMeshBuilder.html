Detailed Description

A utility used to construct dynamically generated meshes, and render them to a FPrimitiveDrawInterface.

Note: This is meant to be easy to use, not fast. It moves the data around more than necessary, and requires dynamically allocating RHI resources. Exercise caution.

Parameters

DepthPriorityGroup- The depth priority group to render the mesh in.
