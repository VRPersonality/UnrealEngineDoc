int32 InitialEnumeratorIndex,
int32 TargetIndex )
static

Reorder enumerators in enum.

Moves the enumerator at the given initial index to a new target index, shifting other enumerators as needed. E.g. with enum [A, B, C, D, E], moving index 1 to index 3 results in [A, C, D, B, E].
