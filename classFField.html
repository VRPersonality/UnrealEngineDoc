◆ GetAuthoredName()

FString FField::GetAuthoredName ( ) const

Returns a human readable string that was assigned to this field at creation.

By default this is the same as GetName() but it can be overridden if that is an internal-only name. This name is consistent in editor/cooked builds, is not localized, and is useful for data import/export.
