enum class FFuzzyPathMatcher::EPathTestPolicy : uint8
strong

The algorithm used to test path matches for a fuzzy path.

Defaults to FString::MatchesWildcard

Enumerator
MatchesWildcard 

Performs the path test with FString::MatchesWildcard.

This is the default algorithm to use for fuzzy paths that can't be optimized with FString::StartsWith.

StartsWith 

Uses FSTring::StartsWith to perform the path test against this fuzzy path.

This is an optimization for fuzzy paths that only contain a single wildcard and the * wildcard only exists at the end of the fuzzy path.
