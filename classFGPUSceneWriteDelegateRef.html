Detailed Description

Abstract reference to a GPUSceneWriteDelegate, such that implementation details can be left to the renderer.

Only contains a raw pointer and performs no life-time management, instead it relies on being constructed by FGPUSceneWriteDelegate which will use AllocateOneFrameResource to ensure a lifetime compatible with mesh batches.
