Detailed Description

A single task for calculating the landscape physical material.

The work uses the GPU so this object handles reading back the data without stalling. We need to call Tick() until IsComplete() returns true. We need to call Release() after use to prevent leaking of internal objects. All calls are expected to be made on the game thread only.
