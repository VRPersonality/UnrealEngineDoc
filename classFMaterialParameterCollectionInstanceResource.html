Detailed Description

Rendering thread mirror of UMaterialParameterCollectionInstance.

Stores data needed to render a material that references a UMaterialParameterCollection.

bool bRecreateUniformBuffer )

Update the contents of the uniform buffer, called from the game thread.

bRecreateUniformBuffer must be false if any components have their rendering state created which might reference this! The calling code is responsible for issuing a FGlobalComponentRecreateRenderStateContext before modifying. Otherwise cached mesh draw commands will be left with a dangling pointer.
