Detailed Description

A Network traffic analyzer that keeps track of useful information including the RTT and uplink bandwidth of the latest received packet to help the congestion control module work.

It uses seq and ack mechanism to track packet and use timestamp and packet size to calculate RTT and bandwidth.

Circular queue that keeps track of useful data of all unacked packets.

Invariant: The tail of the circular queue is the first unacked packet whose packet id is OutAckedSeq + 1. The head of the circular queue is the latest sent packet whose id is equal to OutSeq.
