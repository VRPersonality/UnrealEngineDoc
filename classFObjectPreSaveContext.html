◆ IsProceduralSave()

bool FObjectPreSaveContext::IsProceduralSave ( ) const
inline

Return whether the package is being saved due to a procedural save.

Any save without the possibility of user-generated edits to the package is a procedural save (Cooking, EditorDomain). This allows us to execute transforms that only need to be executed in response to new user data.
