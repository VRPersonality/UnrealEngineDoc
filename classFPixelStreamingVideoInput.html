Detailed Description

The input of the Pixel Streaming system.

Frames enter the system when OnFrame is called. The input buffers might need to be converted/adapted to suit the selected video encoder. CreateAdaptProcess should create a PixelStreamingFrameAdapterProcess that does this work.
