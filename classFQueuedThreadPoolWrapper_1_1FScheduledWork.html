void FQueuedThreadPoolWrapper::FScheduledWork::Abandon ( )
inlineoverrideprivatevirtual

Tells the queued work that it is being abandoned so that it can do per object clean up as needed.

This will only be called if it is being abandoned before completion. NOTE: This requires the object to delete itself using whatever heap it was allocated in.

Implements IQueuedWork.
