virtual

Attemp to reimport all specified objects.

This function will verify that all source file exist and ask the user to decide on how the system will handle asset with missing source file path. Choice are:

  1. Browse every missing source file path before starting the import process
  2. Skip all asset that have a missing source file path
  3. Cancel the whole reimport command
  • Parameters
    ToImportObjectsObjects to try reimporting
  • Parameters
    bShowNotificationTrue to show a notification when complete, false otherwise
  • Parameters
    SourceFileIndexwhich source file index you want to reimport default is INDEX_NONE(the factory will chooseP)
  • Parameters
    bAutomatedTrue to skip dialog prompts
