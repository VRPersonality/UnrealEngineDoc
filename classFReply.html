Detailed Description

A Reply is something that a Slate event returns to the system to notify it about certain aspect of how an event was handled.

For example, a widget may handle an OnMouseDown event by asking the system to give mouse capture to a specific Widget. To do this, return FReply::CaptureMouse( NewMouseCapture ).

◆ CancelFocusRequest()

FReply & FReply::CancelFocusRequest ( )

An event should rarely invoke FReply::CancelFocusRequest.

This will change the reply so that it no longer contains a request for the system to set or clear focus. This is most useful when a reply has been cached for use in async operations.
