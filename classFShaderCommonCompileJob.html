◆ bReleased

uint8 FShaderCommonCompileJob::bReleased

true if the results of the shader compile have been released from the FShaderCompilerManager.

After a job is bFinalized it will be bReleased when ReleaseJob() is invoked, which means that the shader compile thread is no longer processing the job; which is useful for non standard job handling (Niagara as an example).
