◆ RequiredBones

TArray<FBoneIndexType> FSkeletalMeshLODModel::RequiredBones

Bones that should be updated when rendering this LOD.

This may include bones that are not required for rendering. All parents for bones in this array should be present as well - that is, a complete path from the root to each bone. For bone LOD code to work, this array must be in strictly increasing order, to allow easy merging of other required bones.
