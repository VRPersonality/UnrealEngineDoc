FORCEINLINE const FSkeletalMeshLODRenderData * FSkeletalMeshRenderData::GetPendingFirstLOD ( int32 MinLODIdx) const
inline

Return the pending first LOD that can be used for rendering starting at MinLODIdx.

This takes into account the streaming status from PendingFirstLODIdx, and MinLODIdx is expected to be USkeletalMesh::MinLOD, which is platform specific.
