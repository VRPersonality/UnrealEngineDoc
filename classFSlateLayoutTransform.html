Detailed Description

Represents a 2D transformation in the following order: scale then translate.

Used by FGeometry for it's layout transformations.

Matrix form looks like: [Vx Vy 1] * [ S 0 0 ] [ 0 S 0 ] [ Tx Ty 1 ]
