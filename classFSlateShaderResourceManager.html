FVector2f LocalSize,
float DrawScale )
virtual

Creates a handle to a Slate resource A handle is used as fast path for looking up a rendering resource for a given brush when adding Slate draw elements This can be cached and stored safely in code.

It will become invalid when a resource is destroyed It is expensive to create a resource so do not do it in time sensitive areas

Parameters
BrushThe brush to get a rendering resource handle
LocalSizeThe unscaled size of the draw element using this brush. This size is used to rescale vector graphics only
DrawScaleThe scale of the draw element using this brush. This scale is used to rescale vector graphics only
Returns
The created resource handle.
