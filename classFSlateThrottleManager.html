◆ DisableThrottle()

void FSlateThrottleManager::DisableThrottle ( bool bState)

Explicitly disable Slate throttling.

This is intended to be used for code-driven exemptions to Slate throttling such as interactive actions that require multiple refreshes & world ticks. For single refresh scenarios, consider FEditorViewportClient.Invalidate(false, false).

Parameters
bStatetrue if throttling should be disabled.
