◆ bFlushToGPUPending

bool FStaticMeshInstanceBuffer::bFlushToGPUPending
private

If true, then we have updates to the host data not yet committed to the GPU.

This in turn means that bDeferGPUUpload is true, and the Proxy is expected to either call FlushGPUUpload() OR never use the instance data buffers (either is fine).
