const FStaticMeshLODResources * FStaticMeshRenderData::GetCurrentFirstLOD ( int32 MinLODIdx) const
inline

Return the current first LOD that can be used for rendering starting at MinLODIdx.

This takes into account the streaming status from CurrentFirstLODIdx, and MinLODIdx is expected to be UStaticMesh::MinLOD, which is platform specific.
