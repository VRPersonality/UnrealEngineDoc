Detailed Description

The same as FTickableEditorObject, but for systems that need to be ticked periodically during cooking.

If a system needs to be cooked both during cook commandlet and in editor without the cook commandlet, it should dual-inherit from both FTickableCookObject and FTickableEditorObject.
