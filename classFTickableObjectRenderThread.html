Detailed Description

This class provides common registration for render thread tickable objects.

It is an abstract base class requiring you to implement the Tick() method.

virtual bool FTickableObjectRenderThread::IsTickable ( ) const
pure virtual

Pure virtual that must be overloaded by the inheriting class.

It is used to determine whether an object is ready to be ticked. This is required for example for all UObject derived classes as they might be loaded async and therefore won't be ready immediately.

Returns
true if class is ready to be ticked, false otherwise.
