◆ UnwrapWorldPositionToExternalUV()

FVector2f FUVEditorUXSettings::UnwrapWorldPositionToExternalUV ( const FVector3d & VertPosition)
static

Converts from point in unwrap world space, to UV value as displayed to user or seen before import.

This is useful for labeling UV values in the unwrap world, even though the UV values are stored slightly differently inside the mesh itself.

Like other world-UV conversion functions, clients should not know about the details of this conversion.

◆ UVToVertPosition()

FVector3d FUVEditorUXSettings::UVToVertPosition ( const FVector2f & UV)
static

Convert from UV as stored on a mesh in Unreal to world position in the UV editor unwrap world.

This allows changes in UVs of a mesh to be mapped to its unwrap representation.

Like other world-UV conversion functions, clients should not know about the details of this conversion.

References UV.

◆ VertPositionToUV()

FVector2f FUVEditorUXSettings::VertPositionToUV ( const FVector3d & VertPosition)
static

Converts from position in UV editor unwrap world to UV value as stored on a mesh in Unreal.

This allows changes in the unwrap world to be mapped to actual mesh UVs.

Like other world-UV conversion functions, clients should not know about the details of this conversion.
