◆ AlwaysAllowsTranslucentPrimitives()

virtual bool HHitProxy::AlwaysAllowsTranslucentPrimitives ( ) const
inlinevirtual

Method that specifies whether the hit proxy always allows translucent primitives to be associated with it or not, regardless of any other engine/editor setting.

For example, if translucent selection was disabled, any hit proxies returning true would still allow translucent selection.

Returns
true if translucent primitives are always allowed with this hit proxy; false otherwise
