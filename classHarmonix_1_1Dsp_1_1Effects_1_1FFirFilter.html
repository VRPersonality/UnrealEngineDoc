Detailed Description

Finite Impulse Response filter, or (FIR) Filter, that can be used to perform filtering and other signal processing operations.

The filter is defined by a set of "taps" which are coefficients that determine the shape of the filter's frequency response.

virtual void Harmonix::Dsp::Effects::FFirFilter::Init ( const float * InTaps,
int32 InTapCount,
bool InCopyTaps )
virtual

Initializes the filter with the specified taps, which must have InTapCount elements.

If InCopyTaps is true, the taps will be copied into a new buffer owned by the filter; otherwise, the filter will simply store a pointer to the input array.

Any existing tap or history buffers will be freed.

Parameters
InTapsthe array of filter taps
InTapCountthe number of filter taps
InCopyTapswhether to copy the taps into a new buffer
