IAISightTargetInterface::GENERATED_IINTERFACE_BODY ( )
inlineprivate

The method needs to check whether the implementer is visible from given observer's location.

This version allows an asynchronous answer by returning Pending and using the provided Delegate when the information is computed

Parameters
ContextStruct containing all the information required to perform the query
OutSeenLocationThe first visible target location
OutNumberOfLoSChecksPerformedThe number of synchronous queries done. This is used to stop the perception tick sooner when too many LoS checks are done
OutNumberOfAsyncLosCheckRequestedThe number of asynchronous queries done. This is used to stop the perception tick sooner when too many asynchronous LoS checks are done
OutSightStrengthThe sight strength for how well the target is seen
UserDataIf available, it is a data passed between visibility tests for the users to store whatever they want
DelegateIf available, the delegate to call later on, if the result needed asynchronous checks. Calling this delegate within the CanBeSeenFrom call will ensure
Returns
Undefined to indicate that we need to try another CanBeSeenFrom implementation, Visible/NotVisible if visible/not visible from the observer's location, Pending if the check is not finished
