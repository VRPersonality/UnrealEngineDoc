Detailed Description

This interface represents a chunk of VT data allocated and owned by the renderer module, backed by both a page table texture, and a physical texture cache for each layer.

Both page table and physical texture may be shared amongst many different allocated virtual textures. Any method that deals with physical texture requires an explicit LayerIndex parameter to identify the physical texture in question, methods that don't have LayerIndex parameter refer to properties shared by all textures using the given page table These are created with IRendererModule::AllocateVirtualTexture, and destroyed with IRendererModule::DestroyVirtualTexture They must be allocated from render thread, but may be destroyed from any thread
