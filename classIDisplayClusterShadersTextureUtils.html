Custom implementation of an action between input and output contexts.

Iterate over all contexts on input and output textures. Note: This function creates unique TextureContexts instances for the iterator arguments. Multiple calls to ForEachContextByPredicate() will create unique contexts for each use.

Parameters
InSettings- (in) render settings
TextureContextIteratorFunc- predicate function that processes pairs of input and output textures (via resolve or any other way).
