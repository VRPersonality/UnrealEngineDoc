◆ RequestSourceShutdown()

virtual bool ILiveLinkSource::RequestSourceShutdown ( )
pure virtual

Request the source to shutdown.

This may be called multiple times. Should return true, when the source can be destroyed. RequestSourceShutdown will be called multiple times until it returns true. If the source is multithreaded, the source should test if it can be shutdown, set a flag and return immediately.
