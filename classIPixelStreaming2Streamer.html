Detailed Description

The IPixelStreaming2Streamer interface provides functionality for managing a Pixel Streaming session.

This class manages the core operations of streaming, such as setting the framerate, handling video input, and interacting with the signalling server. It also allows for control over connected players and sending data to them.

virtual void IPixelStreaming2Streamer::SetCoupleFramerate ( bool bCouple)
pure virtual

Setting this to true will cause the streamer to ignore the FPS value and instead push out frames as they are submitted from the video input.

If the encode process takes longer than the time between frames, frames will be dropped.

Parameters
bCoupletrue to couple the streamer framerate to that of the video input.
