Detailed Description

Systems that implement a reload capability implement this interface and register it with the module manager while a reload is in progress.

◆ GetReinstancedCDO() [2/2]

virtual UObject * IReload::GetReinstancedCDO ( UObject * CDO)
pure virtual

For a given CDO, return the new CDO if it has been re-instanced.

If it hasn't been re-instanced, then the supplied CDO will be returned. If a non-CDO object is passed in, it will be returned as is.

Implemented in FReload.
