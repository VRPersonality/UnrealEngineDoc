Detailed Description

FMetaSoundParameterTransmitter provides a communication interface for sending values to a MetaSound instance.

It relies on the send/receive transmission system to ferry data from the transmitter to the MetaSound instance. Data will be safely ushered across thread boundaries in scenarios where the instance transmitter and metasound instance live on different threads.
