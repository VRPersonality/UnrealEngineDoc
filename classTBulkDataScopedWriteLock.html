Detailed Description

template<typename ElementType>
class TBulkDataScopedWriteLock< ElementType >

Locks the given bulkdata object for read/write access.

Note that performing actions on the original bulkdata object while this scope is active (such as calls to realloc) may cause problems. Only use this object for bulkdata operations while it is active.
