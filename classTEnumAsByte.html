Detailed Description

template<class InEnumType>
class TEnumAsByte< InEnumType >

Template to store enumeration values as bytes in a type-safe way.

Traits classes and constants which test if a type is a TEnumAsByte.

Blueprint enums should either be enum classes (preferred): enum class EMyEnum : uint8 { One, Two }, which doesn't require wrapping in this template or a namespaced enum namespace EMyEnum { enum Type // <- literally Type, not a placeholder { One, Two }; }
