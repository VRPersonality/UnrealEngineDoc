Detailed Description

template<int32 SIZE, int TPaddingForCacheContention, typename TTrackingCounter = FNoopCounter>
class TLockFreeFixedSizeAllocator< SIZE, TPaddingForCacheContention, TTrackingCounter >

Thread safe, lock free pooling allocator of fixed size blocks that only returns free space when the allocator is destroyed.

Alignment isn't handled; assumes FMemory::Malloc will work.
