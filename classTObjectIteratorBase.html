Detailed Description

template<EObjectIteratorThreadSafetyOptions IteratorThreadSafety>
class TObjectIteratorBase< IteratorThreadSafety >

Class for iterating through all objects, including class default objects.

Note that when Playing In Editor, this will find objects in the editor as well as the PIE world, in an indeterminate order. IteratorThreadSafety template parameter manages how this class handles thread safety of UObject iteration.
