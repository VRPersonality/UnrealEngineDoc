Detailed Description

template<typename VersionObjectType, typename ValueType>
class TRigVMModelCachedValue< VersionObjectType, ValueType >

A cached value which relies on a version providing object.

The versioned object needs to provide uint32 GetCachedValueVersion() Values within the cache are considered valid if they are a) set and b) the version of the cache matches the version of the versioned object.
