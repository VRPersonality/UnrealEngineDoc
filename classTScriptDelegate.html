◆ ProcessDelegate()

template<typename InThreadSafetyMode>
template<class UObjectTemplate>
void TScriptDelegate< InThreadSafetyMode >::ProcessDelegate ( void * Parameters) const
inline

Executes a delegate by calling the named function on the object bound to the delegate.

You should always first verify that the delegate is safe to execute by calling IsBound() before calling this function. In general, you should never call this function directly. Instead, call Execute() on a derived class.
