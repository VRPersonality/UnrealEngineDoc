Detailed Description

template<class ObjectType, ESPMode InMode>
class TSharedPtr< ObjectType, InMode >

TSharedPtr is a non-intrusive reference-counted authoritative object pointer.

This shared pointer will be conditionally thread-safe when the optional Mode template argument is set to ThreadSafe.
