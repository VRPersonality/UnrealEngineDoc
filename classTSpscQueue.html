Detailed Description

template<typename T, typename AllocatorType = FMemory>
class TSpscQueue< T, AllocatorType >

Fast single-producer/single-consumer unbounded concurrent queue.

Doesn't free memory until destruction but recycles consumed items. Based on http://www.1024cores.net/home/lock-free-algorithms/queues/unbounded-spsc-queue
