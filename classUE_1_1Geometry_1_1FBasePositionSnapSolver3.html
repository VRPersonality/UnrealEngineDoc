Detailed Description

FBasePositionSnapSolver3 is a base class for 3D position snapping implementations.

It is not usable on its own and the split between this class and the implementations is not incredibly clean. However there is lots of shared functionality that is placed here.

3D Point and Line targets are supported. Targets can also be "ignored".

The actual snap "solve" must be implemented by subclasses, depending on their input data (ray, point, line, etc)
