Detailed Description

FDynamicMeshChange stores a "change" in a FDynamicMesh3, which in this context means the replacement of one set of triangles with a second set, that may have different vertices/attributes.

The change can be applied and reverted via Apply()

Construction of a well-formed FDynamicMeshChange is quite complex and it is strongly suggested that you do so via FDynamicMeshChangeTracker
