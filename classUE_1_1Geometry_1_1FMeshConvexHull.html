Choose a more or less evenly-spaced subset of mesh vertices.

Conceptually, this function creates a uniform grid with given cell size. (Cell size is given roughly as a percentage of the total mesh bounding box.) Each grid cell can hold up to one vertex. Return the set of representative vertices, maximum one per cell.

Parameters
MeshSurface mesh whose vertices we want to sample
CellSizeAsPercentOfBoundsGrid cell size expressed as a percentage of the mesh bounding box size
OutSamplesIndices of chosen vertices
