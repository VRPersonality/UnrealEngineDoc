void UE::Geometry::FMeshMapBaker::SetTargetMeshUVCharts ( TArray< int32 > * UVChartsIn)
inline

Set an a Triangle ID to UV Chart ID array for TargetMesh.

If this is not set, then the baker will compute it as part of Bake(). Since ComputeUVCharts() is non-trivial, this method is intended to allow a client to externally cache the result of ComputeUVCharts to minimize the overhead per bake.

Parameters
UVChartsInthe TriID to UVChartID map

For evaluators using EAccumulateMode::Overwrite the default is used when weight == 0, and the value is used otherwise.
