Detailed Description

Walk the surface of an FDynamicMesh to try find a planar path connecting two points.

Paths include every vertex and edge they need to cross. Greedy algorithm will only return one path if there are multiple. Represent a path on the surface of a mesh via barycentric coordinates and triangle references
