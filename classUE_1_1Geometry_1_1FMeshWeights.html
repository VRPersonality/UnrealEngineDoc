static

Compute cotan-weighted blend for a vertex one-ring, with some weight analysis/clamping to avoid vertices getting "stuck" in explicit integration/iterations.

If failure is detected, Uniform centroid is returned, which does cause some tangential flow. Equivalent to CotanCentroidSafe() if the weights are used to blend vertex positions, but can be used to weight other properties.

Parameters
BlendingFuncBlendingFunc(NbrVertexID, Weight) will be called for every one-ring neighbour vertex
DegenerateTolif any weights are larger than this value, return uniform weights instead. Should be > 1.
bFailedToUniformwill be set to true if non-null and result was clamped
