Detailed Description

Used to linearize the VtxIds in a mesh as a single array and allow mapping from array offset to mesh VtxId.

Generally, the array offset will correspond to a matrix row when forming a Laplacian. The last NumBoundaryVerts are the boundary verts. This may be zero.
