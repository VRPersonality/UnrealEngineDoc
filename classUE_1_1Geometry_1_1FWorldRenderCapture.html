Detailed Description

FWorldRenderCapture captures a rendering of a set of Actors in a World from a specific viewpoint.

Various types of rendering are supported, as defined by ERenderCaptureType. Currently rendering an entire World, ie without an explicit list of Actors or Components, is not supported.
