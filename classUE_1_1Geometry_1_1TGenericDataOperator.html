Detailed Description

template<typename ResultType>
class UE::Geometry::TGenericDataOperator< ResultType >

TGenericDataOperator is a base interface for operator implementations that can produce an object of arbitrary data type.

Ownership is unique, ie the Operator calculates the result and then the caller takes it via ExtractResult()
