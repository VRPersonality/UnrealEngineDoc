Detailed Description

template<class PointSetType>
class UE::Geometry::TIncrementalMeshDijkstra< PointSetType >

TIncrementalMeshDijkstra computes graph distances on a mesh from seed point(s) using Dijkstra's algorithm.

Derived from TMeshDijkstra. This Incremental variant allows adding new Seed points to an already-computed solution.
