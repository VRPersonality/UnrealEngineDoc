Detailed Description

template<typename RealType>
class UE::Geometry::TNormalHistogram< RealType >

TNormalHistogram calculates/represents a histogram on a 3D sphere.

The histogram bins are defined by a Spherical Fibonacci distribution, ie each bin is the voronoi cell of the Fibonacci point. This point set can be calculated for any N.

One current limitation of this implementation is that the unit +/- X/Y/Z axes are not bin center points.
