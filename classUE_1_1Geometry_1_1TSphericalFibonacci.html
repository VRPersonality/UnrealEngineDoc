Detailed Description

template<typename RealType>
class UE::Geometry::TSphericalFibonacci< RealType >

A Spherical Fibonacci (SF) Point Set is a set of points that are roughly evenly distributed on a sphere.

The points lie on a spiral, see https://dl.acm.org/doi/10.1145/2816795.2818131 for more information. The i-th SF point an N-point set can be calculated directly. For a given (normalized) point P, finding the nearest SF point (ie mapping back to i) can be done in constant time.

Z is "up" in the sphere.
