◆ AllTargetLayerNames

TArray<FName> UE::Landscape::EditLayers::FMergeContext::AllTargetLayerNames
protected

List of all target layer names that can possibly be rendered on this landscape (even invalid ones).

The index of the target layer in that list is important : it allows to use a TBitArray instead of a TSet to designate a list of target layers all throughout the merge, which greatly accelerates all the intersection tests being made. If merging heightmaps, it contains a single name (that is only meaningful for debug display).
