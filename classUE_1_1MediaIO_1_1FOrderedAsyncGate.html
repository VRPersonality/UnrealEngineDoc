◆ GiveUpTurn()

void UE::MediaIO::FOrderedAsyncGate::GiveUpTurn ( const uint32 Number)
inline

Call this function when you are ready to give up your turn.

It must be called with the number received from GetANumber, and it must be called whether WaitForTurn was called or not. Not giving up the turn will prevent any others from getting their turn. Note: It will internally call WaitForTurn.

◆ WaitForTurn()

void UE::MediaIO::FOrderedAsyncGate::WaitForTurn ( const uint32 Number) const
inline

This function will block until it is the turn of the given number to execute.

It can be called multiple times until the turn is given up by calling GiveUpTurn. The number given must have been received from a call to GetANumber.
