Detailed Description

Extension Class to add Navigation Tool Columns.

Example #1: This would create a Tag Column at the end of the current column list (order matters!) ColumnExtender.AddColumn<FNavigationToolTagColumn>();

Example #2: This would create a Tag Column before the Label Column (if it doesn't exist, it's the same behavior as the above example) ColumnExtender.AddColumn<FNavigationToolTagColumn, ENavigationToolExtensionPosition::Before, FNavigationToolLabelColumn>();
