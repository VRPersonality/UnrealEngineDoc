bool UE::StereoRenderUtils::FStereoShaderAspects::IsMobileMultiViewEnabled ( ) const
inline

Whether mobile multiview is enabled - i.e.

using VK_KHR_multiview. Another drawcall reduction technique, independent of instanced stereo. Mobile multiview generates view indices to index into texture arrays. Can be internally emulated using instanced stereo if native support is unavailable, by using ISR-generated view indices to index into texture arrays.
