◆ GetErrorFormatString()

virtual const FText UE::UnifiedError::IErrorDetails::GetErrorFormatString ( const FError & Error) const
pure virtual

GetErrorFormatString; specifies the default error format string to be used when generating FError::GetErrorMessage.

The format string can specify any property exposed by any encapsulated IErrorDetails::GetErrorProperties. Example: GetErrorProperties adds Name:"ModuleId" Value:10. GetErrorFormatString returns "Module id was {ModuleId}". Result "Module id was 10".
