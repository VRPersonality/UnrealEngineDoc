UParticleModuleSubUVMovie::UPROPERTY ( )
private

The starting image index for the SubUV (1 = the first frame).

Assumes order of Left->Right, Top->Bottom If greater than the last frame, it will clamp to the last one. If 0, then randomly selects a starting frame.

GENERATED_UCLASS_BODY() UPROPERTY() uint32 bUseEmitterTime UParticleModuleSubUVMovie::UPROPERTY ( )

If true, use the emitter time to look up the frame rate.

If false (default), use the particle relative time. The frame rate the SubUV images should be 'flipped' thru at.
