Detailed Description

Mechanic for selecting elements of a dynamic mesh in the UV editor.

Interacts heavily with UUVToolSelectionAPI, which actually stores selections.

bool UUVEditorMeshSelectionMechanic::RaycastCanonicals ( const FRay & WorldRay,
bool bRaycastIsForUnwrap,
bool bPreferSelected,
FRaycastResult & HitOut ) const

This is a helper method that doesn't get used in normal selection mechanic operation, but can be used by clients if they need to raycast canonical meshes, since the mechanic already keeps aabb trees for them.

The mechanic does not need to be enabled to use this (and typically will not be, if a tool is doing its own raycasting).
