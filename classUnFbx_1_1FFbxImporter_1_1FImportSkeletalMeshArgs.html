Detailed Description

Creates a skeletal mesh from Fbx Nodes with the given name and flags, imported from within the FBX scene.

These Fbx Nodes bind to same skeleton. We need to bind them to one skeletal mesh.

Parameters
InParent
NodeArrayFbx Nodes to import
Namethe Unreal Mesh name after import
Flags
FbxShapeArrayFbx Morph objects.
OutData- Optional import data to populate
bCreateRenderData- Whether or not skeletal mesh rendering data will be created.
OrderedMaterialNamesIf not null, the original fbx ordered materials name will be use to reorder the section of the mesh we currently import
Returns
The USkeletalMesh object created
