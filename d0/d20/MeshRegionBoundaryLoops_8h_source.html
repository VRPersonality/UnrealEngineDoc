82 /**
83 * Generates a map from vertex ID's of the loop vertices to corresponding overlay element ID's

85 * edge goes out of that vertex in the loop direction.
86 * The function is useful in cases where the triangles inside the loop will be deleted and replaced,
87 * and we want to keep the same UV's on the border vertices.
88 * Does not clear LoopVidsToOverlayElementsOut before use (simply adds to it), so could
