FSlateWindowElementList::STAT ( int32 MemManagerAllocatedMemory; )

The window to render to.

This may be different from the paint window if we are displaying the contents of a window (or virtual window) onto another window The primary use case of this is thread safe rendering of widgets during times when the main thread is blocked (e.g loading movies) If this is null, the paint window is used
