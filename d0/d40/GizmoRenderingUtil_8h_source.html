65 /**
66 * Gets a custom material suitable to use for gizmo components. The material is drawn on top of opaque geometry with
67 * dithering for portions behind opaque materials, but uses the custom depth buffer to properly occlude itself.
68 * Components using this material should set bRenderCustomDepth to true so they can occlude other gizmo elements. It
69 * is also suggested that TranslucencySortPriority be set to something like UE::GizmoRenderingUtil::GIZMO_TRANSLUCENCY_SORT_PRIORITY
70 * so that the component is drawn on top of other translucent materials.
71 *
72 * @param Outer Object to set as outer for the material instance. Typically can be left as nullptr to use transient package.
73 */
