enum enum VIS_ModeratelyAggressive VIS_MostAggressive VIS_Max enum enum VLM_SparseVolumeLightingSamples UMETA ( DisplayName = "Multiply Scale With Root Component Scale")

Multiply value of the SpawnTransform Parameter with the default scale in the actor's root component.

Volume lighting samples are placed on top of static surfaces at medium density, and everywhere else in the Lightmass Importance Volume at low density.

Positions outside of the Importance Volume will have no indirect lighting. This method requires CPU interpolation so the Indirect Lighting Cache is used to interpolate results for each dynamic object, adding Rendering Thread overhead.
Volumetric Fog cannot be affected by precomputed lighting with this method.
