36 /** Copies skin weight attribute from one mesh to another. Assumes the two geometries are identical or near-identical.
37 * Uses closest triangle on the target mesh to interpolate skin weights to each of the points on the target mesh.
38 * Attributes for the given profiles on both meshes should exist in order for this function to succeed.
39 * @param InSourceMesh The mesh to copy skin weights from.
40 * @param InTargetMesh The mesh to copy skin weights to.
41 * @param InSourceProfile The name of the skin weight profile on the source mesh to read from.
42 * @param InTargetProfile The name of the skin weight profile on the target mesh to write to.
43 * @param SourceBoneIndexToTargetBoneIndexMap An optional mapping table to map bone indexes on the source mesh to the target mesh.
44 * The table needs to be complete for all the source bone indexes to valid target bone indexes, otherwise the behavior
45 * is undefined. If the table is not given, the bone indexes on the source and target meshes are assumed to be the same.

55 /** Remaps the bone indices on all skin weight attributes from one index to another. The array view should contain a full mapping of all the
56 * bone indices contained in the skin weights. The array is indexed by the current bone index and the value at that index is the new bone index.
57 * If the mapping is incomplete or if two entries map to the same bone, the result is undefined. No prior checking is performed.
58 * @param InMesh The mesh on which to modify all skin weight attributes to remap their bone indices.
59 * @param InBoneIndexMapping The mapping from one bone index to another. The old bone index is used to index into the array, the value at that
60 * position is the new bone index.
61 * @return true if the operation was successful. If the mapping array was incomplete then this will return false. If there are no skin weight
62 * attributes on the mesh, then the operation is still deemed successful.
