Recomputes the key position OutputValues as if the points were mirrored accross an imaginary x-axis aligned mirror edge.

The mirror Edge is positioned between the top and bottom edge.

Note
This function only recalculates key positions. Tangents must be set separately.
Parameters
InKeysToMirrorKeys located between BottomHeight and TopHeight, whose tangents will be recomputed.
InCurveIdThe curve the keys belong to
InBottomHeightThe height of the bottom edge.
InTopHeightThe height of the top edge.
InMirrorHeightThe height of the imaginary x-axis-aligned mirror edge across which tangents are mirrored.
InCurveEditorReference to the curve editor used to set key attributes.
