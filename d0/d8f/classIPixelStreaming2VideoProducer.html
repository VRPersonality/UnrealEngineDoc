Detailed Description

A "Video Producer" is an object that you use to push video frames into the Pixel Streaming system.

Example usage:

(1) Each new frame you want to push you call MyVideoProducer->PushFrame(MyFrame)

Note: Your frame is likely a FPixelCaptureInputFrameRHI if the frame is a GPU texture or a FPixelCaptureInputFrameI420 or FPixelCaptureInputFrameNV12 if your frame is from the CPU.
