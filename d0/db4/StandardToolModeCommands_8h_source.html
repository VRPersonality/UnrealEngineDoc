27/**
28 * FStandardToolModeCommands provides standard commands for Tools. This allows
29 * for "sharing" common hotkey bindings between multiple EditorModes
30 *
31 * The set of standard commands is defined by EStandardToolModeCommands.
32 * You must call FStandardToolModeCommands::Register() on module startup to register these commands
33 * (note that they may already be registered by another module, but you should call this to be safe).
34 *
35 * Then, when you want to Bind a standard command in your FUICommandList,
36 * call FStandardToolModeCommands::Get().FindStandardCommand() to get the registered UICommandInfo.
37 *
38 */
