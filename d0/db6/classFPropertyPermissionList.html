void FPropertyPermissionList::SetEnabled ( bool bEnable,

Turn on or off the property PermissionList.

DoesPropertyPassFilter will always return true if disabled. Optionally the change can be announced through a broadcast, which is typically recommended, but may not be needed if the permission list is only temporary enabled, e.g. during saving.
