Detailed Description

template<typename ViewModelType, bool bChecked>
struct UE::Sequencer::TImplicitViewModelCastImpl< ViewModelType, bChecked >

Proxy type that enables implicit dynamic casting of a view-model to any other view-model or extension type.

Shared/weak extension pointer forward declarations - include MVVM/ViewModelPtr.h for complete declarations.

This is implemented as a proxy type rather than the conversion operators existing on FViewModelPtr so that the implicit cast must be user-directed, while still removing additional template boilerplate at the call-site.

See also
FViewModelPtr::ImplicitCast
