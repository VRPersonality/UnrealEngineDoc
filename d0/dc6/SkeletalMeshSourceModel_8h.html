Returns true if there is mesh description data available on this bulk data object. This says nothing about the validity of the actual mesh description data, only that it exists. It's can be used as a quick way to verify without unpacking the bulk data into an accessible mesh description object.

Create a new mesh description object on this container, obliterating whatever mesh might have been there before. Does not invalidate the bulk data unless CommitMeshDescription is called. Use this if wanting to start afresh.

Gets the cached mesh description, if there was one, otherwise tries to load it from the bulk data. If there is no bulk data, returns nullptr.
