Detailed Description

Base class of the custom render pass.

Create a derived class to provide custom behavior. Custom render pass is rendered as part of the main render loop.

strong
Enumerator
NoCapture 

Don't trigger a render capture.

Capture 

Start a render capture on PreRender, stop it on PostRender.

BeginCapture 

Start a render capture on PreRender (this must be followed by another pass with EndCapture).

Use it when the render capture encapsulates several custom passes

EndCapture 

Stop a render capture on PostRender (this must be preceded by another pass with BeginCapture).

Use it when the render capture encapsulates several custom passes
