77/**
78 * Abstract interface that defines all sequencer interactions for any channel type
79 * Channels are stored internally as FMovieSceneChannel*, with this interface providing a common set of operations for all channels through a safe cast from the FMovieSceneChannel*.
80 * Implementations are found in TSequencerChanelInterface which calls overloaded free functions for each channel.
