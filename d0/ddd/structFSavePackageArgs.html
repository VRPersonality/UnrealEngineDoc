Detailed Description

Struct to encapsulate UPackage::Save arguments.

These arguments are shared between packages when saving multiple packages concurrently.

◆ TopLevelFlags

EObjectFlags FSavePackageArgs::TopLevelFlags = RF_NoFlags

For all objects which are not referenced[either directly, or indirectly] through the InAsset provided to the Save call (See UPackage::Save), only objects that contain any of these flags will be saved.

If RF_NoFlags is specified, only objects which are referenced by InAsset will be saved into the package.
