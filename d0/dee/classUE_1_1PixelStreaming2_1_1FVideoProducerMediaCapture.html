static

Creates a video input where the user can specify their own MediaIO output and capture.

This method does not does not configure or start capturing, this is left to the user. Use this constructor if you know how to configure the MediaIOCapture yourself or don't want to capture the active viewport.

Parameters
MediaCaptureThe custom MediaIOCapture that will pass its captured frames as video input.
Returns
A video input backed by the passed in MediaIO capture.
