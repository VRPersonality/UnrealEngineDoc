46/**
47 * UPlanePositionGizmo implements a gizmo interaction where 2D parameter value is manipulated
48 * by dragging a point on a 3D plane in space. The 3D position is converted to 2D coordinates
49 * based on the tangent axes of the plane.
50 *
51 * As with other base gizmos, this class only implements the interaction. The visual aspect of the
52 * gizmo, the plane, and the parameter storage are all provided externally.
53 *
54 * The plane is provided by an IGizmoAxisSource. The origin and normal define the plane and then
55 * the tangent axes of the source define the coordinate space.
56 *
57 * The interaction target (ie the thing you have to click on to start the dragging interaction) is provided by an IGizmoClickTarget.
58 *
59 * The new 2D parameter value is sent to an IGizmoVec2ParameterSource
60 *
61 * Internally a UClickDragInputBehavior is used to handle mouse input, configured in ::Setup()
62 */
