46/**
47 * UAxisPositionGizmo implements a gizmo interaction where 1D parameter value is manipulated
48 * by dragging a point on a 3D line/axis in space. The 3D point is converted to the axis parameter at
49 * the nearest point, giving us the 1D parameter value.
50 *
51 * As with other base gizmos, this class only implements the interaction. The visual aspect of the
52 * gizmo, the axis, and the parameter storage are all provided externally.
53 *
54 * The axis direction+origin is provided by an IGizmoAxisSource.
55 *
56 * The interaction target (ie the thing you have to click on to start the dragging interaction) is provided by an IGizmoClickTarget.
57 *
58 * The new 1D parameter value is sent to an IGizmoFloatParameterSource
59 *
60 * Internally a UClickDragInputBehavior is used to handle mouse input, configured in ::Setup()
61 */
