◆ NumTextureLayers

uint8 FVTProducerDescription::NumTextureLayers = 0u

Producers will fill a number of texture layers.

These texture layers can be distributed across one or more physical groups. Each physical group can contain one or more of the texture layers. Within a physical group the texture layers share the same UV allocation/mapping and can be referenced by a single page table lookup.
