18/**
19 * CanvasRenderTarget2D is 2D render target which exposes a Canvas interface to allow you to draw elements onto
20 * it directly. Use CreateCanvasRenderTarget2D() to create a render target texture by unique name, then
21 * bind a function to the OnCanvasRenderTargetUpdate delegate which will be called when the render target is
22 * updated. If you need to repaint your canvas every single frame, simply call UpdateResource() on it from a Tick
23 * function. Also, remember to hold onto your new canvas render target with a reference so that it doesn't get
24 * garbage collected.
25 */

35 /**
36 * Updates the the canvas render target texture's resource. This is where the render target will create or
37 * find a canvas object to use. It also calls UpdateResourceImmediate() to clear the render target texture
38 * from the deferred rendering list, to stop the texture from being cleared the next frame. From there it
39 * will ask the rendering thread to set up the RHI viewport. The canvas is then set up for rendering and
40 * then the user's update delegate is called. The canvas is then flushed and the RHI resolves the
41 * texture to make it available for rendering.
