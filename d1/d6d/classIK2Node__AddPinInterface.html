Detailed Description

Interface for adding the small "Add Pin" symbol to a node in the bottom right hand side.

Implementing this interface will provide the API needed to get the UI up and running, but the actual pin creation/naming is up to the specific node.

See also
UK2Node_CommutativeAssociativeBinaryOperator, UK2Node_DoOnceMultiInput
