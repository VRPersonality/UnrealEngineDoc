Detailed Description

Trait Pointer A trait pointer represents a shared pointer to allocated instance data.

It manages reference counting. It points to a FTraitInstanceData or FNodeInstance when resolved.

A node pointer can also be weak, meaning that it does not update reference counting. Note that weak pointers will point to garbage if the trait instance is destroyed.

Weak trait pointers should be used carefully!

See also
FNodeInstance, FTraitInstanceData
