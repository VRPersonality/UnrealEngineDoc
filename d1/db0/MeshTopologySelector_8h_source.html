61/**
62 * FMeshTopologySelector implements selection behavior for a subset of mesh elements.
63 * Topological Groups, Edges, and Corners can be selected depending on the subclass of FTopologyProvider used.
64 *
65 * Internally a FGeometrySet3 is constructed to support ray-hit testing against the edges and corners.
66 *
67 * Note that to hit-test against the mesh you have to provide a your own FDynamicMeshAABBTree3.
68 * You do this by providing a callback via SetSpatialSource(). The reason for this is that
69 * (1) owners of an FMeshTopologySelector likely already have a BVTree and (2) if the use case
70 * is deformation, we need to make sure the owner has recomputed the BVTree before we call functions
71 * on it. The callback you provide should do that!
72 *
73 * DrawSelection() can be used to visualize a selection via line/circle drawing.
74 *
75 * @todo optionally have an internal mesh AABBTree that can be used when owner does not provide one?
76 */
77

165 /**
166 * Given a camera rectangle (from a marquee selection) gives the corners/edges/groups that are contained in
167 * the rectangle. Only selects one of the types, and prefers corners to edges to groups.
168 *
169 * For group edges, occlusion is checked at endpoints, and if either is occluded, the edge is considered occluded.
170 * For faces, occlusion is checked at triangle centroids.
171 *
172 * @param Settings Settings that determine what is selected and whether occlusion is tested.
173 * @param CameraRectangle The camera rectangle.
174 * @param TargetTransform Transform from points to world space, since the rectangle is in world space.
175 * @param ResultOut Output selection (cleared before use).
176 * @param TriIsOccludedCache Cached values of whether a given Tid has an occluded centroid, since occlusion
177 * values will stay the same if the rectangle is resized as long as the camera is not moved. This should
178 * be used when doing occlusion-based marquee face selection on meshes with many triangles, because the
179 * occlusion checks are expensive.
180 *
181 * @return true if something was selected.
182 */
183 UE_API bool FindSelectedElement(const FSelectionSettings& Settings, const FCameraRectangle& CameraRectangle,
