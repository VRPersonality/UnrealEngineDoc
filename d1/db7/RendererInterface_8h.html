RENDERCORE_API bool CacheRenderTarget ( FRHITexture * Texture,
const TCHAR * Name,
TRefCountPtr< IPooledRenderTarget > & OutPooledRenderTarget )
extern

Creates an untracked pooled render target from the RHI texture, but only if the pooled render target is empty or doesn't match the input texture.

If the pointer already exists and points at the input texture, the function just returns. Useful to cache a pooled render target for an RHI texture. Returns true if the render target was created, or false if it was reused.
