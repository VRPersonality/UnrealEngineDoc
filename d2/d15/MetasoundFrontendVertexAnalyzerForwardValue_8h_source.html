18 // Templatized implementation of the simplest/cheapest form of an analyzer,
19 // which just forwards a given value to an associated view or views. Can be
20 // used in conjunction with an additional process, task, etc. handled by a
21 // view to avoid further processing/expense within the graph execution thread
22 // context.
