◆ ReadTexture()

MODELINGCOMPONENTS_API bool UE::AssetUtils::ReadTexture ( UTexture2D * TextureMap,
TImageBuilder< FVector4f > & DestImageOut,
const bool bPreferPlatformData = false )

Extract the Dimensions and image pixels from an input TextureMap By default, the "Source" texture data is read.

This is only available in-Editor. At runtime, only "Platform" data is available.

Parameters
TextureMapthe texture map to read
DestImageOutthe result of the texture read
bPreferPlatformDataif true, platform data will be returned even in-Editor.
Returns
true on success
