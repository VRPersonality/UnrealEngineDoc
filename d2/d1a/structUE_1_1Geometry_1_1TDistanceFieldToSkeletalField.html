Detailed Description

template<typename InputBoundedImplicitType, typename RealType>
struct UE::Geometry::TDistanceFieldToSkeletalField< InputBoundedImplicitType, RealType >

This class converts the interval [-falloff,falloff] to [0,1], Then applies Wyvill falloff function (1-t^2)^3.

The result is a skeletal-primitive-like shape with the distance=0 isocontour lying just before midway in the range (at the .ZeroIsocontour constant)
