Detailed Description

template<typename Converter, int32 DefaultConversionSize = 128u>
class TStringConversion< Converter, DefaultConversionSize >

Class takes one type of string and converts it to another.

The class includes a chunk of presized memory of the destination type. If the presized array is too small, it mallocs the memory needed and frees on the class going out of scope.
