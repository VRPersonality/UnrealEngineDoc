UParticleModuleLocationPrimitiveCylinder::UPROPERTY ( )
private

Determine the particle system axis that should represent the height of the cylinder.

Can be one of the following: PMLPC_HEIGHTAXIS_X - Orient the height along the particle system X-axis. PMLPC_HEIGHTAXIS_Y - Orient the height along the particle system Y-axis. PMLPC_HEIGHTAXIS_Z - Orient the height along the particle system Z-axis.
