Detailed Description

template<typename COLL_DATA_PROVIDER, typename KDOP_IDX_TYPE>
struct TkDOPLineCollisionCheck< COLL_DATA_PROVIDER, KDOP_IDX_TYPE >

This struct holds the information used to do a line check against the kDOP tree.

The collision provider gives access to various matrices, vertex data etc. without having to use virtual functions.

Initializes all of the variables that are used throughout the line check.
