Detailed Description

Complete implementation of a curve editor tree.

Only really defines the hierarchy and selection states for tree items.

ECurveEditorTreeFilterState InheritedState )
private

Run the specified filters over the specified items and their recursive children, storing the results in this instance's FilterStates struct.

Parameters
FilterPtrsArray of non-null pointers to filters to use. Items are considered matched if they match any filter in this array.
ItemsToFilterArray item IDs to filter
InheritedStateThe filter state for each item to receive if it does not directly match a filter (either ECurveEditorTreeFilterState::NoMatch or ECurveEditorTreeFilterState::InheritedChild)
Returns
Raised state flags which should be applied to parents of the filtered items.
