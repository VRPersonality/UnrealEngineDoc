259 /**
260 * We draw up to 2 horizontal value indicator line on the keys that a user selects.
261 * This makes it easier for the user to see a selected key's value in relation to other keys.
262 *
263 * Requirements:
264 * - Only draw value indicator line if 1 curve is selected. More curves selected = no line.
265 * - The value indicator lines are drawn through the min and max value keys in the selected range.
266 * - If only 1 key is selected, only draw 1 line (obviously).
267 * The per-curve upper limit of value indicator lines that are drawn is configured in UCurveEditorSettings::MaxNumHorizontalSelectionLinesPerCurve.
268 *
269 * This is called in OnPaint, which is const. Mutable is a workaround.
270 */
