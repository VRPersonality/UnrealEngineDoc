Add a key-value association to the map.

The association doesn't replace any of the key's existing associations. However, if both the key and value match an existing association in the map, no new association is made and the existing association's value is returned.

Parameters
InKeyThe key to associate.
InValueThe value to associate.
Returns
A reference to the value as stored in the map; the reference is only valid until the next change to any key in the map.
