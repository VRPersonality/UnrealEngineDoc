16 /**
17 * The Type of LOD in a Mesh Asset. Note some options are only applicable to some asset types (e.g., only Static Meshes)
18 */
19 enum class EMeshLODType : uint8
20 {
21 /** The Maximum-quality available SourceModel LOD (HiResSourceModel if it is available, otherwise SourceModel LOD0) */
22 MaxAvailable,
23 /** The HiRes SourceModel. LOD Index is ignored. HiResSourceModel is not available at Runtime. */
24 HiResSourceModel,
25 /**
26 * The SourceModel mesh at a given LOD Index. Note that a StaticMesh Asset with Auto-Generated LODs may not have a valid SourceModel for every LOD Index
27 * SourceModel meshes are not available at Runtime.
28 */
29 SourceModel,
30 /**
31 * The Render mesh at at given LOD Index.
32 * A StaticMesh Asset derives its RenderData LODs from it's SourceModel LODs. RenderData LODs always exist for every valid LOD Index.
33 * However the RenderData LODs are not identical to SourceModel LODs, in particular they will be split at UV seams, Hard Normal creases, etc.
34 * RenderData LODs in a StaticMesh Asset are only available at Runtime if the bAllowCPUAccess flag was enabled on the Asset at Cook time.
35 */
36 RenderData
37 };
