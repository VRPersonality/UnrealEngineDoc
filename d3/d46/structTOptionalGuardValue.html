Detailed Description

template<typename RefType, typename AssignedType = RefType>
struct TOptionalGuardValue< RefType, AssignedType >

exception-safe guard around saving/restoring a value.

Commonly used to make sure a value is restored even if the code early outs in the future. Usage: TOptionalGuardValue<bool> GuardSomeBool(bSomeBool, false); // Sets bSomeBool to false, and restores it in dtor.
