template<typename ElementType, int32 NodeCapacity = 4>
TArray<FNode> TQuadTree< ElementType, NodeCapacity >::Nodes
private

Contains the actual elements this tree is responsible for.

Nodes are used to keep track of each element's AABB as well. For a non-internal leaf, this is the list of nodes that are fully contained within this tree. For an internal tree, this contains the nodes that overlap multiple subtrees.
