bool IPackageWriter::FCapabilities::bDeclareRegionForEachAdditionalFile = false

Whether an entry should be created for each BulkData stored in the BulkData section This is necessary for some Writers that need to be able to load the BulkDatas individually.

For other writers the extra regions are an unnecessary performance cost.
