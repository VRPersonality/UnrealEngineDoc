Detailed Description

template<typename T, uint8 N>
struct UE::MovieScene::TMultiChannelValue< T, N >

Generic value type that supports a specific number of channels, optionally masked out.

Used for blending types that can be represented as a contiguous set of numeric values. Relative, Weighted and Additive blending occurs on a per-channel basis
