35/**
36 * FMetaNavMeshPath allows creating hierarchical or guided navmesh paths
37 *
38 * Waypoints array defines list of locations that will be used to create actual FNavMeshPath data during path following.
39 * On path following start and upon reaching waypoint, path will request update from owning navmesh with following parameters:
40 * - start location set to agent location
41 * - end location set to target waypoint (or goal actor for last one)
42 * - goal actor set only for last section
43 *
44 * Since path updates itself for moving agent it really shouldn't be reused by others.
45 */
