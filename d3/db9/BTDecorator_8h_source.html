19/**
20 * Decorators are supporting nodes placed on parent-child connection, that receive notification about execution flow and can be ticked
21 *
22 * Because some of them can be instanced for specific AI, following virtual functions are not marked as const:
23 * - OnNodeActivation
24 * - OnNodeDeactivation
25 * - OnNodeProcessed
26 * - OnBecomeRelevant (from UBTAuxiliaryNode)
27 * - OnCeaseRelevant (from UBTAuxiliaryNode)
28 * - TickNode (from UBTAuxiliaryNode)
29 *
30 * If your node is not being instanced (default behavior), DO NOT change any properties of object within those functions!
31 * Template nodes are shared across all behavior tree components using the same tree asset and must store
32 * their runtime properties in provided NodeMemory block (allocation size determined by GetInstanceMemorySize() )
33 *
34 */
