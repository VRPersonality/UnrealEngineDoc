Detailed Description

Rendering resources needed to render an individual static mesh LOD.

This structure is ref counted to allow the LOD streamer to evaluate the number of readers to it (readers that could access the CPU data). Because the stream out clears the CPU readable data, CPU code that samples it must ensure to only reference LODs above CurrentFirstLODIdx.
