void FPlatformTypeLayoutParameters::AppendKeyString ( FString & KeyString) const

Allow the layout parameters to modify the given DDC key string.

Since layout parameters are part of e.g. material shadermap ID, they should result in two different DDC entries for two different IDs, even if binary layouts happen to be compatible.
