Detailed Description

template<typename DataType>
struct Metasound::TDataTypeLiteralFactory< DataType >

TDataTypeLiteralFactory creates a DataType.

TDataTypeLiteralFactory provides several factory methods for forwarding arguments from the factory method to the DataType constructor. See TDataFactory for more information on the provided factory methods.
