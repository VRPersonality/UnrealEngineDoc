bool bColors )

Fast path to update various vertex buffers.

This path does not support changing the size/counts of any of the sub-buffers, a direct memcopy from the CPU-side buffer to the RHI buffer is used.

Warning
This can only be called on the Rendering Thread.

This path rebuilds all the resources and reconfigures the vertex factory, so the counts/etc could be modified.
