14 /**
15 * Trait Pointer
16 * A trait pointer represents a shared pointer to allocated instance data.
17 * It manages reference counting.
18 * It points to a FTraitInstanceData or FNodeInstance when resolved.
19 *
20 * A node pointer can also be weak, meaning that it does not update reference counting.
21 * Note that weak pointers will point to garbage if the trait instance is destroyed.
22 *
23 * Weak trait pointers should be used carefully!
24 *
25 * @see FNodeInstance, FTraitInstanceData
26 */
