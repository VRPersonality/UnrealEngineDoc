◆ ScalingAdjustmentLimit

double UE::Geometry::FExtrudeBoundaryEdges::ScalingAdjustmentLimit = 1.0

If greater than 1, maximal amount by which a vertex can be moved in an attempt to keep edges parallel to original edges while extruding.

This "movement" is done by scaling the X axis of the extrude frame.
