44 /**
45 * Returns true if the profile curve is counterclockwise relative to the rotation direction. I.e.,
46 * if you are looking from the side with the revolution axis pointing up and the curve to the left of
47 * the axis, the curve should be counterclockwise.
48 * If the curve is not closed, the function looks at the bottom of the profile curve relative to the
49 * revolution axis and checks whether the outer edge of the bottom goes in the direction that it would on
50 * a counterclockwise curve (up, in our example).
51 *
52 * Results are undefined if a fully revolved mesh would be self intersecting.
53 * Assumes that RevolutionAxisDirection is normalized.
54 */
55 GEOMETRYCORE_API
56 bool ProfileIsCCWRelativeRevolve(TArray<FVector3d>& ProfileCurve, const FVector3d& RevolutionAxisOrigin,
57 const FVector3d& RevolutionAxisDirection, bool bProfileCurveIsClosed);
