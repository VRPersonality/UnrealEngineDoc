virtual void IMappedFileRegion::PreloadHint ( int64 PreloadOffset = 0,
int64 BytesToPreload = MAX_int64 )
inlinevirtual

Synchronously preload part or all of the mapped region.

Typically this is done by reading a byte from each CPU page. This is only a hint, some platforms might ignore it. There are not guarantees how long this data will stay in memory.

Parameters
PreloadOffsetOffset into this region to preload
BytesToPreloadnumber of bytes to preload. This is clamped to the size of the mapped region
