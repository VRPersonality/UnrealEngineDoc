78 /**
79 * Indicates whether CDOs should be included in reference replacement.
80 * Disabled by default since this can increase search cost, because this
81 * effectively means replacement will require us to do a referencer search
82 * for at least one instance of every remapped class (i.e. - the CDO). In
83 * most cases (e.g. load time), incurring this cost is unnecessary because
84 * the CDO is not expected to be referenced outside of its own class type,
85 * so we treat it as an opt-in behavior rather than enabling it by default.
86 *
87 * Note: This flag is implied to be 'true' if 'OriginalCDO' is non-NULL. In
