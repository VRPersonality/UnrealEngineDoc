Detailed Description

template<typename DataType>
class Metasound::TLiteralOperatorLiteralFactory< DataType >

TExecutableLiteralOperator is used for executable types.

The majority of literals are constant values, but executable data types cannot be constant values.

Note: this is supporting for FTrigger literals which are deprecated. TLiteralOperatorLiteralFactory creates an input by passing it a literal.
