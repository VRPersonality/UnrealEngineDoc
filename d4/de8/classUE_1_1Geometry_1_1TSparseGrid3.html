Detailed Description

template<typename ElemType>
class UE::Geometry::TSparseGrid3< ElemType >

Dynamic sparse 3D grid.

Useful in cases where we have grid of some type of non-trivial object and we don't want to pre-allocate full grid of them. So we allocate on-demand. This can be used to implement multi-grid schemes, eg for example the ElemType could be sub-grid of fixed dimension.

Todo
