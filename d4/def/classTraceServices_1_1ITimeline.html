Finds event information for the event closest to InTime from the interval [InTime - DeltaTime, InTime + DeltaTime].

Parameters
InTime- The time used to query for the event
DeltaTime- Events from interval [InTime - DeltaTime, InTime + DeltaTime] will be considered. The one closest to InTime will be returned
Depth- The Depth used to query for the event
Returns
True if an event was found, False if no event was found for the specified input parameters
