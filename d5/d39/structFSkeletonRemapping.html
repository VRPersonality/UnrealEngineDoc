void FSkeletonRemapping::ComposeWith ( const FSkeletonRemapping & OtherSkeletonRemapping)

Compose this remapping with another remapping in place.

The other remapping's source skeleton must match this remapping's target skeleton. The result will map from this remapping's source skeleton to the other remapping's target skeleton
