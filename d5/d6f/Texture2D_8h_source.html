205 /**
206 * Retrieve initial texel data for mips, starting from FirstMipToLoad, up to the last mip in the texture.
207 * This function will only return mips that are currently loaded, or available via derived data.
208 * THIS FUNCTION WILL FAIL if you include in the requested range a streaming mip that is NOT currently loaded.
209 * Also note that ALL mip bulk data buffers are discarded (either returned or freed).
210 * So... in short, this function should only ever be called once.
211 *
212 * @param FirstMipToLoad - The first mip index to load.
213 * @param OutMipData - A pre-allocated array of pointers, that correspond to [FirstMipToLoad, .... LastMip]
214 * Upon successful return, each of those pointers will point to allocated memory containing the corresponding mip's data.
215 * Caller takes responsibility to free that memory.
216 * @param OutMipSize - A pre-allocated array of int64, that should be the same size as OutMipData (or zero size if caller does not require the sizes to be returned)
217 * Upon successful return, each element contains the size of the corresponding mip's data buffer.
218 * @returns true if the requested mip data has been successfully returned.
219 */
220 virtual bool GetInitialMipData(int32 InFirstMipToLoad, TArrayView<void*> OutMipData, TArrayView<int64> OutMipSize);
