15/**
16 * The IPixelStreaming2Module interface manages the core functionality of the Pixel Streaming system.
17 * This class provides access to streamers, video and audio producers, and handles starting and stopping the streaming process.
18 * It also allows interaction with the signalling server and manages the lifecycle of streamers within the Pixel Streaming system.
19 */
