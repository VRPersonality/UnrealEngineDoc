Detailed Description

template<typename T, int32 ArraySize>
class FMovingWindowAverageFast< T, ArraySize >

This class calculates a moving window average.

Its designed to be used with floats or doubles and keeps track of the average with every value pushed so is ideal when there is a one to one or one to many relationship between calls to PushValue() and GetMovingWindowAverage() respectively.
