void UE::Geometry::FMeshEdgeSelection::ExpandToOneRingNeighbors ( TFunctionRef< bool(int32)> FilterF)
inline

Add all one-ring neighbors of current selection to set.

On a large selection this is quite expensive as we don't know the boundary, so we have to iterate over all selected edges.

Return false from FilterF to prevent vertices from being included.
