Detailed Description

template<typename OccluderTriangleMeshType>
class UE::Geometry::TRemoveOccludedTriangles< OccluderTriangleMeshType >

Remove "occluded" triangles, i.e.

triangles on the "inside" of the mesh(es). This is a fuzzy definition, current implementation has a couple of options including a winding number-based version and an ambient-occlusion-ish version, where if face is occluded for all test rays, then we classify it as inside and remove it.

Note this class always removes triangles from an FDynamicMesh3, but can use any mesh type to define the occluding geometry (as long as the mesh type implements the TTriangleMeshAdapter fns)
