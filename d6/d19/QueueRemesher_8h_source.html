14/**
15 *
16 * Extension to Remesher that is smarter about which edges/vertices to touch:
17 * - A buffer tracks edges that were affected on last pass, and hence might need to be updated
18 * - FastSplitIteration() just does splits, to reach target edge length as quickly as possible
19 * - RemeshIteration() applies remesh pass for modified edges
20 * - TrackedFullSmoothPass_Buffer() smooths all vertices but only adds to queue if edge changes enough
21 * - TrackedFullProjectionPass() projects all vertices but only adds to queue if edge changes enough
22 *
23 */
