11/**
12 * FStructView is "typed" struct pointer, it contains pointer to struct plus UScriptStruct pointer.
13 * FStructView does not own the memory and will not free it when out of scope.
14 * It should be only used to pass struct pointer in a limited scope, or when the user controls the lifetime of the struct being stored.
15 * E.g. instead of passing ref or pointer to a FInstancedStruct, you should use FConstStructView or FStructView to pass around a view to the contents.
16 * FStructView is passed by value.
17 * FStructView is similar to FStructOnScope, but FStructView is a view only (FStructOnScope can either own the memory or be a view)
18 * const FStructView prevents the struct from pointing at a different instance of a struct. However the actual struct
19 * data can be mutated. Use FConstStructView to prevent mutation of the actual struct data.
20 * See FConstStructView for examples.
21 */

23{
24public:

184///////////////////////////////////////////////////////////////// FConstStructView /////////////////////////////////////////////////////////////////
185/**
186 * FConstStructView is "typed" struct pointer, it contains const pointer to struct plus UScriptStruct pointer.
187 * FConstStructView does not own the memory and will not free it when of scope.
188 * It should be only used to pass struct pointer in a limited scope, or when the user controls the lifetime of the struct being stored.
189 * E.g. instead of passing ref or pointer to a FInstancedStruct, you should use FConstStructView or FStructView to pass around a view to the contents.
190 * FConstStructView is passed by value.
191 * FConstStructView is similar to FStructOnScope, but FConstStructView is a view only (FStructOnScope can either own the memory or be a view)
192 * FConstStructView prevents mutation of the actual struct data however the struct being pointed at can be changed to point at a different instance
193 * of a struct. To also prevent this use const FConstStructView.
194 * e.g. FStructView A; FConstStructView B = A; // compiles
195 * e.g. FStructView A; FConstStructView B; B = A; // compiles as B can be made to point at any other StructView
196 * e.g. FConstStructView A; FStructView B = A; // doesn't compile as the struct data for A is immutable (but mutable for B)
197 * e.g. FStructView A; A.Foo = NewVal; // compiles as the struct data for A is mutable.
198 * e.g. FConstStructView A; A.Foo = NewVal; // doesn't compile as the struct data for A is immutable.
199 * e.g. FStructView A; const FStructView B; A = B; // compiles as the struct B is pointing to can't be made to point at something else but A isn't const.
200 * e.g. const FStructView A; FStructView B; A = B; // doesn't compile as attempting to make const view point at something else
201 */
