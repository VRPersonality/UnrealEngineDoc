84/**
85 * Transfer float properties from a source collection to a target collection.
86 * Component Transfer is used when all geometries from the source collection have matched names with the target collection.
87 * Otherwise, Global Transfer is used.
88 * Geometries are matched when the geometry's BoneName can be found as the start of the BoneName of a geometry in the target collection.
89 * Use TransformNameSuffix to add extra string to the source geometry's BoneName to avoid multiple matched names.
90 * For example, source geometry has name SK_10 and target geometry has name SK_10_tet1
91 * For all names, Check BoneName attribute in Transform group in the collection.
92 */
