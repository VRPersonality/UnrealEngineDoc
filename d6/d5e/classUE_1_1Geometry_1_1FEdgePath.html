Detailed Description

FEdgePath is is little more than a double link-list that exposes IDs instead of node pointers.

The ID of a removed segment may be reused for a subsequently appended or inserted segment. When used in conjunction with a mesh, it represents a directed, edge-aligned path.

Note: this exists primarily for use with FDeformableEdgePath and no checking is done to ensure the path is continuous, in fact it could be used to represent multiple disconnected paths.
