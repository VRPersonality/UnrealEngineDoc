36/**
37 * Base class of UnrealEd brush builders.
38 *
39 *
40 * Tips for writing brush builders:
41 *
42 * - Always validate the user-specified and call BadParameters function
43 * if anything is wrong, instead of actually building geometry.
44 * If you build an invalid brush due to bad user parameters, you'll
45 * cause an extraordinary amount of pain for the poor user.
46 *
47 * - When generating polygons with more than 3 vertices, BE SURE all the
48 * polygon's vertices are coplanar! Out-of-plane polygons will cause
49 * geometry to be corrupted.
