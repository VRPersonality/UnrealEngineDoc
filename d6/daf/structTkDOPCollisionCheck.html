template<typename COLL_DATA_PROVIDER, typename KDOP_IDX_TYPE>
struct TkDOPCollisionCheck< COLL_DATA_PROVIDER, KDOP_IDX_TYPE >

Base struct for all collision checks.

Holds a reference to the collision data provider, which is a struct that abstracts out the access to a particular mesh/primitives data
