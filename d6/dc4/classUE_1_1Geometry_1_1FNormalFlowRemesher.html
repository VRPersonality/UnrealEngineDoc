Detailed Description

Remeshing with "face aligned projection".

This approach to projection attempts to preserve sharp features in the mesh by aligning triangle normals with normals from the original mesh during the projection pass. This class also performs a pass of edge flips aimed at further aligning triangle normals to their originals. This can be helpful in fixing the occasional single "bad edge" along a sequence of feature edges.

DYNAMICMESH_API bool UE::Geometry::FNormalFlowRemesher::EdgeFlipWouldReduceNormalError ( int EdgeID,
double BadEdgeErrorThreshold = 0.01,
double ImprovementRatioThreshold = 0.75 ) const
protected

Test to see if flipping an edge would improve "normal error".

Normal error for a triangle is: 0.5 * (1.0 - TriangleNormal \cdot ProjectedNormal), where ProjectedNormal is the normal of the triangle returned by finding the closest point on ProjTarget. The normal error for an edge is the sum of errors for the edge's incident triangles.

Parameters
EdgeIDEdge to consider.
BadEdgeErrorThresholdOnly edges with this error or greater are candidates for flipping.
ImprovementRatioThresholdNew edge must have error less than old edge error times this ratio.
Returns
True iff (CurrentEdgeError > BadEdgeErrorThreshold) and (NewEdgeError < ImprovementRatioThreshold * CurrentEdgeError)
