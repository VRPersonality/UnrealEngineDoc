Detailed Description

Defines a sort batch that regroups several sort tasks created by FGPUSortManager::AddTask(), becoming sort the batch elements.

The condition to regroup task in the same batch rely mostly around the key precisions and the sort location (PreRender or PostRenderOpaque). Each SortBatch maps to a single GPU sort task.
