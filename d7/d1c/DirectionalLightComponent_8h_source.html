82 /**
83 * Proportion of the fade region between cascades.
84 * Pixels within the fade region of two cascades have their shadows blended to avoid hard transitions between quality levels.
85 * A value of zero eliminates the fade region, creating hard transitions.
86 * Higher values increase the size of the fade region, creating a more gradual transition between cascades.
87 * The value is expressed as a percentage proportion (i.e. 0.1 = 10% overlap).
88 * Ideal values are the smallest possible which still hide the transition.
89 * An increased fade region size causes an increase in shadow rendering cost.
90 */
