enum class UE::Geometry::FLinearExtrusionOp::ESelectionShapeModifierMode
strong

Modifier applied to the selected triangles during extrusion.

Enumerator
None 

No modification, keep the same shape.

FlattenToPlane 

Project the selection vertices to the XY plane of StartFrame before extruding.

RaycastToPlane 

Cast a ray for each selection vertex against plane defined by ToFrame's Orgin/Z-Axis and use the hit position.

◆ bInferGroupsFromNeighbours

bool UE::Geometry::FLinearExtrusionOp::bInferGroupsFromNeighbours = true

Assign new groups to regions of the extrusion "tube" based on the existing adjacent groups before extrusion.

Each (disconnected) unique pair of group IDs gets a new unique group ID on the extrusion. If disabled, the entire extrusion tube will be assigned a single new group ID
