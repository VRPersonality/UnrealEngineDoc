Detailed Description

Manages level actor viewport selectability and hovered visual states.

Contains static methods to enable outside modules to implement their own management.

Selects or deselects actors in a world.

If no actors are specified, uses all the actors in the level.

Parameters
InPredicateFunction to use to check if the actor should be selected/deselected
InActorsOptional list of actors to selected/deselected
bInSelectIf true, selects the actors. If false, deselects the actors
bInClearSelectionIf true, clears the current selection before selecting the new actors
Returns
True if atleast one new actor was selected/deselected
