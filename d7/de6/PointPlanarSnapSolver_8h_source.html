16/**
17 * FPointPlanarSnapSolver solves for a Point snap location on a plane, based
18 * on an input Point and a set of target points and lines in the plane.
19 *
20 * This implementation has the notion of a "history" of previous points,
21 * from which line and distance constraints can be inferred.
22 * This is useful for snapping in 2D polygon drawing.
23 *
24 * See FBasePositionSnapSolver3 for details on how to set up the snap problem
25 * and get results.
26 */

42 /** How much more important a known length is than its line's priority. */
43 int KnownLengthPriorityDelta = 10;

45 /** How much more important an intersection is than the more important of the intersecting lines. */
46 int32 IntersectionPriorityDelta = 11;
47
48 int MinInternalPriority() const { return FMath::Min(CardinalAxisPriority, LastSegmentPriority) - FMath::Max(IntersectionPriorityDelta, KnownLengthPriorityDelta); }

53 /**
54 * Creates snap lines based on the last point in the point history. Calling the function
55 * with both parameters set to false clears the generated snap lines.
56 *
57 * @param bCardinalAxes If true, the generated lines are parallel to the axes of the plane
58 * and pass through the last point.
59 * @param bLastHistorySegment If true, adds a snap line through the last point that is 90
60 * degrees to the last segment.
61 */
62 UE_API void RegenerateTargetLines(bool bCardinalAxes, bool bLastHistorySegment);
63
64 /**
65 * Sets the snapping lines to be based on the history points *adjacent* to the point with a
66 * given history index. The given index can be one beyond the ends of the current history
67 * (i.e., -1 or PointHistoryLength) to base the lines on the first/last points.
68 * The snap lines will be parallel to the plane axes.
69 * Useful for moving a point to be aligned with one of its neighbors.
70 *
71 * @param HistoryIndex Index in the range [-1, PointHistoryLength()]. The points adjacent
72 * to that index will be used for line generation.
73 * @param bWrapAround If true, the first point will be considered adjacent to the last
74 * when HistoryIndex is 0 or PointHistoryLength()-1.
75 * @param bGenerateIntersections If true, intersections will be generated as higher-priority
76 * points. Intersections are performed with generated and user-specified lines that
77 * lie in the plane.
78 */
79 UE_API void RegenerateTargetLinesAround(int32 HistoryIndex, bool bWrapAround = false, bool bGenerateIntersections = true);
