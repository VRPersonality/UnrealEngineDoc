bool UE::Geometry::FGroupEdgeInserter::FEdgeLoopInsertionParams::bSimplifyAlongPath = true

When inserting edges, we can locally simplify to improve mesh quality along the edge path.

Only does so when it should not change mesh shape, UVs, triangle groups, etc, and only applies when Insertion Mode is "Plane Cut"

◆ Mode

EInsertionMode UE::Geometry::FGroupEdgeInserter::FEdgeLoopInsertionParams::Mode = EInsertionMode::Retriangulate

Determines how the edge is inserted: by using a cutting plane to cut existing triangles along the path, or deleting triangles and retriangulating.

Retriangulation will keep geometry nice but currently loses the UV's, and it obviously breaks for non-planar groups.
