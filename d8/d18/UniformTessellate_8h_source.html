19/**
20 * Given an input mesh and a tessellation level, this operator generates a new tessellated mesh where every triangle in
21 * the input mesh is uniformly subtriangulated into (TessellationNum + 1)^2 triangles. Per-vertex normals, uvs,
22 * colors and extended per-vertex attributes are linearly interpolated to the new vertices. Per-triangle group
23 * identifiers/materials and extended triangle attributes for the new triangles are inherited from the corresponding
24 * input mesh triangles they replaced.
25 *
26 * Note: Currently does not support interpolation of the GenericAttributes besides Skin Weights.
27 *
28 * o o
29 * / \ TessellationNum = 2 / \
30 * / \ / \
31 * / \ x-----x
32 * / \ FUniformTessellate / \ / \
33 * / \ -------------> / \ / \
34 * / \ x-----*-----x
35 * / \ / \ / \ / \
36 * / \ / \ / \ / \
37 * o-----------------o o-----x-----x-----o
38 */
