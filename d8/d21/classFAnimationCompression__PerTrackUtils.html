uint16 FAnimationCompression_PerTrackUtils::CompressFixed16 ( float Value,
int32 Log2MaxValue = 0 )
inlinestatic

Compresses a float into a signed fixed point number, which can range from the symmetrical values of -2^Log2MaxValue .

. 2^Log2MaxValue. No clamping is done, values that don't fit will overflow.

For example, a Log2MaxValue of 0 can encode -1..1, and 7 can encode -128..128.

Parameters
ValueValue to encode
Log2MaxValueEncoding range (can be 0..15)
Returns
The quantized value
