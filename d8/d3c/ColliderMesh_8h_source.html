23/**
24 * FColliderMesh is a minimal representation of an Indexed Triangle Mesh suitable to use
25 * with a TMeshAABBTree3. This class is intended to be used in situations where a copy
26 * of a FDynamicMesh3 would be created/kept only to use with an AABBTree or FWNTree. In such
27 * situations, the FDynamicMesh3 copy is (relatively) heavy memory-wise, particularly if a large
28 * number of small mesh/AABBTree pairs are stored. In those cases a FColliderMesh can be used instead.
29 *
30 * If FBuildOptions.bBuildAABBTree is true (default), then an AABBTree will automatically be
31 * built and available via GetAABBTree()
32 *
33 * If the source FDynamicMesh3 is not compact, it will be compacted. ID mappings can optionally
34 * be stored if the FBuildOptions are configured to request it (default false), allowing
35 * hit-triangle/vertex IDs to be mapped back to IDs on the source mesh.
36 *
37 * Currently there is no support for UVs or Vertex Normals.
38 *
39 * FColliderMesh (effectively) implements the TTriangleMeshAdapter interface, and so can be used
40 * anywhere a TTriangleMeshAdapter could be used (eg with TMeshQueries functions).
41 */
