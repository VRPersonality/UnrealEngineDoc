38/**
39 * UPointSetComponent is a Component that draws a set of points, as small squares.
40 * Per-point Color and (view-space) Size is supported. Normals are not supported.
41 *
42 * Points are inserted with an externally-defined ID, internally this is done via
43 * a TSparseArray. This class allocates a contiguous TArray large enugh to hold the
44 * largest ID. Using ReservePoints() may be beneficial for huge arrays.
45 *
46 * The points are drawn as two triangles (ie a square) orthogonal to the view direction.
47 * The actual point size is calculated in the shader, and so a custom material must be used.
48 */
