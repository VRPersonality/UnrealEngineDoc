91 * If called on a component instance in a world (and the class is changing), the created ChildActor will use the class defaults as template.
92 * @param InClass The Actor subclass to spawn as a child actor

100 /**
101 * Sets then class to use for the child actor providing an optional Actor to use as the template.
102 * If called on a template component (owned by a CDO) and NewChildActorTemplate is not null, the new child actor template will be created using the supplied Actor as template.
103 * If called on a template component and NewChildActorTemplate is null, the properties of any existing child actor template will be copied as best possible to the template.
104 * If called on a component instance in a world with NewChildActorTemplate not null, then if registered a new child actor will be created using the supplied Actor as template,
105 * otherwise if not registered it will ensure. If the class also changed, then future ChildActors created by this component the class defaults will be used.
106 * If called on a component instance in a world with NewChildActorTemplate null and the class is changing, the created ChildActor will use the class defaults as template.
107 * @param InClass The Actor subclass to spawn as a child actor
108 * @param NewChildActorTemplate An Actor to use as the template when spawning a child actor using this component (per the rules listed above)
109 */
