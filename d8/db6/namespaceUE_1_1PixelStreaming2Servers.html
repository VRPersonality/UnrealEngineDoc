PIXELSTREAMING2SERVERS_API TSharedPtr< IServer > UE::PixelStreaming2Servers::MakeSignallingServer ( )

Creates a native C++ signalling server (similar to cirrus.js) with no dependencies launched inside the Unreal Engine process.

Note: Calling this method does not launch the server. You should call Launch() yourself once you have bound to appropriate delegates such as OnReady.

Returns
The embedded signalling server.
