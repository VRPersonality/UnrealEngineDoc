void UE::Geometry::FMeshVertexSelection::ExpandToOneRingNeighbours ( const TUniqueFunction< bool(int)> & FilterF = nullptr)
inline

Add all one-ring neighbours of current selection to set.

On a large selection this is quite expensive as we don't know the boundary, so we have to iterate over all selected vertices.

Return false from FilterF to prevent vertices from being included.
