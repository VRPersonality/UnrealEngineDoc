Disabling this option will keep degenerate triangles found.

In general you should leave this option on. For static meshes, enabling this option will allow you to use Nanite rendering at runtime. Can only be used with simple opaque materials. If checked, one convex hull per UCX_ prefixed collision mesh will be generated instead of decomposing into multiple hulls If checked, collision will automatically be generated (ignored if custom collision is imported or used). For static meshes, enabling this option will combine all meshes in the FBX into a single monolithic mesh in Unreal Scale to apply to the mesh when allocating the distance field volume texture. The default scale is 1, which is assuming that the mesh will be placed unscaled in the world.
