DoNotRunInsideBusyWait 

Tells the scheduler if this task is allowed to run during another task's busy wait.

The default should be true for most case but it is sometime useful to avoid it if this task is going to wait on another one, and that other task busy waits, this could cause a cycle that could deadlock. (i.e. T1 -> busywait -> picks T2 that then waits on T1 -> deadlock In this case, we can decide that T2 should never be picked up by busy waits.
