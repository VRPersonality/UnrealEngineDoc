101 /**
102 * Looks through the supplied graph for a node that best matches the one specified.
103 * Sometimes (when diffing separate assets) there could be more than one possible
104 * match, so providing a list of already matched nodes helps us narrow it down (and
105 * prevents us from matching one node with multiple others).
106 *
107 * @param OldGraph The graph you want to search.
108 * @param NewNode The new node you want to match.
109 * @param PriorMatches Previous made matches to exclude from our search.
110 * @return A pair of nodes (including the supplied one) that best match each other (one may be nullptr if no match was found).
111 */

123 /**
