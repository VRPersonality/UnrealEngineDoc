218 /**
219 * Cut existing UV topology with a set of edges. This allows for creating partial seams/darts, interior cuts, etc.
220 *
221 * Avoids creating bowties. In cases where an edge is not next to any present or future seams/borders, some
222 * adjacent edge will be picked to be made into a seam as well, since it's impossible to make the original
223 * into a seam otherwise.
224
225 * @param EidsToMakeIntoSeams list of edges to turn into seams
226 * @param Result if non-null, list of new UV elements created along the path will be stored here (not ordered)
227 * @return true on success
228 */
229 UE_API bool CreateSeamsAtEdges(const TSet<int32>& EidsToMakeIntoSeams, FUVEditResult* Result = nullptr);

231 /**
232 * Takes the currently selected triangles and makes a separate UV island out of them, i.e. any interior seams
233 * are removed, and seams are added around the boundary of the selection. If the selection is not connected
234 * in the mesh, islands will be created for each connected component of selected triangles. If some of the
235 * selected triangles have unset UVs, they will be initialized to per-vertex zero UVs for the purposes of
236 * creating an island.
237 *
238 * @param TidsToMakeIntoIsland Triangles in the mesh
239 * @param Result if non-null, list of new UV elements created for new seams, or to initialize UVs.
240 * @param ChangedTrianglesOut Optional output of triangles that had their UVs changed. Currently this is a conservative
241 * superset, so some might not have actually changed. Not cleared before use.
242 * @return true on success
243 */
244 UE_API bool MakeIsland(const TSet<int32>& TidsToMakeIntoIsland, FUVEditResult* Result = nullptr, TSet<int32>* ChangedTrianglesOut = nullptr);
