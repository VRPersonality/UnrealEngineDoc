Detailed Description

FDynamicMeshUVPacker implements various strategies for packing UV islands in a UV Overlay.

The island topology and UV unwraps must already be created, this class simply scales/rotates/translates the islands to fit.

◆ StackPack()

DYNAMICMESH_API bool UE::Geometry::FDynamicMeshUVPacker::StackPack ( )

Uniformly scale all UV islands so that the largest fits in positive-unit-square, and translate each islands separately so that it's bbox-min is at the origin.

So the islands are "stacked" and all fit in the unit box.

◆ StandardPack()

DYNAMICMESH_API bool UE::Geometry::FDynamicMeshUVPacker::StandardPack ( )

Standard UnrealEngine UV layout, similar to that used for Lightmap UVs.

All UV islands are packed into standard positive-unit-square. Only supports single-pixel border size.
