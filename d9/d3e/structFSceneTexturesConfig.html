Detailed Description

Struct containing the scene texture configuration used to create scene textures.

Use InitializeViewFamily to initialize the SceneTexturesConfig structure in the FViewFamilyInfo. A global singleton instance is maintained manually with static Set / Get functions, but will soon be deprecated, in preference of using the structure from the FViewFamilyInfo.
