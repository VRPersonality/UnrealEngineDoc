Detailed Description

template<typename RealType>
struct UE::Geometry::TQuadricError< RealType >

QuadricError represents a quadratic function that evaluates distance to plane.

Stores minimal 10-coefficient form, following http://mgarland.org/files/papers/qtheory.pdf (symmetric matrix A, vector b, constant c)
