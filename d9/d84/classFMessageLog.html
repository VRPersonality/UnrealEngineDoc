Helper functions to allow easy adding of messages to the log.

Each one will add a token for the severity (if valid) and the textual message (if any). Messages are buffered.

void FMessageLog::NewPage ( const FText & InLabel)

Add a new page to the log.

Pages will not be created until messages are added to them, so we dont generate lots of empty pages. This call will cause a flush so that the logs state is properly reflected.
