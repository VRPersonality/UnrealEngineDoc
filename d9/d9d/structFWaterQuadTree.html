◆ TileRegion

FBox2D FWaterQuadTree::TileRegion
private

This is the bounding box that describes where the water tiles can be generated.

This will be same or smaller than the root node bounds of the tree and might not be square. Shares the same Min position with the root node bounds, so this region is a subset of the root node bounds.
