Detailed Description

Singleton class used for optionally overriding previous transforms for motion vector computation.

This predominantly used by Sequencer on camera cut frames to forcibly inform the renderer of simulated trajectories for primitive components and cameras. Transforms are stored in an unintrusive sparse map per-component to avoid paying a memory cost for all components. Simulated transforms are only valid for the frame that they were added, and are removed on a subsequent tick.
