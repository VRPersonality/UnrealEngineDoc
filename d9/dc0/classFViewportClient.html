Detailed Description

An abstract interface to a viewport's client.

The viewport's client processes input received by the viewport, and draws the viewport.

◆ WindowCloseRequested()

virtual bool FViewportClient::WindowCloseRequested ( )
inlinevirtual

Called when the top level window associated with the viewport has been requested to close.

At this point, the viewport has not been closed and the operation may be canceled. This may not called from PIE, Editor Windows, on consoles, or before the game ends from other methods. This is only when the platform specific window is closed.

Returns
True if the viewport may be closed, false otherwise.
