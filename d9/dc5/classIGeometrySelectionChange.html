Detailed Description

Base interface intended to be implemented by FToolCommandChange subclasses that represent changes to a FGeometrySelection.

The point of the interface is to allow UGeometrySelectionManager to execute the ApplyChange() and RevertChange() with the FGeometrySelectionEditor it provides. So the call pattern is that the FToolCommandChange::Apply(), with the UGeometrySelectionManager as the UObject target, will call UGeometrySelectionManager::ApplyChange(), which then in turn will call IGeometrySelectionChange::ApplyChange()
