void FEditorCacheArchetypeManager::CacheArchetype ( TNotNull< const UObject * > Object,
UObject * Archetype = nullptr )
inline

Caches the archetype for this object during reinstantiation has we rename object in that phase.

This is only a utility method to ensure the archetype is not lost during reinstantiation as it renames and changes outers which makes impossible to retrieve it during the process.

Parameters
Objecttop cache its archetype
Archetypeto use as a cache, if null it will use GetArchetype on the owner of the specified object
