UNREALED_API void FEditorViewportStats::BeginFrame ( )
static

Begins the frame for capturing using statements.

If nothing is logged between the begin and end frame we reset the last using tracking variable in EndFrame.

◆ EndFrame()

UNREALED_API void FEditorViewportStats::EndFrame ( )
static

Use EndEndUsing to manually reset the Using state so that the next call to Using will commit a new record.

Useful when you know a transition has occurred like the mouse button has been released.

UNREALED_API void FEditorViewportStats::Using ( Category InCategory)
static

Using commits a single usage record for whichever category is sent to it only if it's different from the last category that was sent to Using.

This should be used to capture usage data for modes where it's difficult to tell when they ended.
