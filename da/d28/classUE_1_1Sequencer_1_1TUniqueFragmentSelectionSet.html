Detailed Description

template<typename KeyType, typename ModelType>
class UE::Sequencer::TUniqueFragmentSelectionSet< KeyType, ModelType >

Selection set that defines a fragment selection of a view model where each fragment is globally unique.

A good example is FKeyHandle or any other monotomic counter. Selection must be associated with a view model type defined by the template parameter ModelType.
