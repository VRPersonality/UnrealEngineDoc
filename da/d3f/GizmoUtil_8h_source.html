40 /**
41 * Uses the gizmo manager to create a gizmo of the given class (assuming that the gizmo type does not need
42 * any special setup beyond instantiation) without having to register a custom builder for that class ahead of time.
43 *
44 * This function lets the user bypass the need to define, register, and use a builder class, while still registering
45 * the gizmo properly with the gizmo manager. Under the hood, it creates and registers a temporary generic builder,
46 * uses it to make the gizmo, and then immediately deregisters the builder.
47 */

51 /**
52 * Template version of CreateGizmoViaSimpleBuilder that does a cast on return.
