METASOUNDGRAPHCORE_API TPimplPtr< FDirectedGraphAlgoAdapter > Metasound::FDirectedGraphAlgo::CreateDirectedGraphAlgoAdapter ( const IGraph & InGraph)
static

The FDirectedGraphAlgoAdapter caches internal representations of the passed in IGraph.

Using this can reduce redundant calcuations when multiple algorithms are run on the same IGraph (without altering the IGraph).

This function creates the adapter.
