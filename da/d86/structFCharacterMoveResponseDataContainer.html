Detailed Description

Response from the server to the client about a move that is being acknowledged.

Internally it mainly copies the FClientAdjustment from the UCharacterMovementComponent indicating the response, as well as setting a few relevant flags about the response and serializing the response to and from an FArchive for handling the variable-size payload over the network.
