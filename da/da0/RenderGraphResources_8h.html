Enumeration Type Documentation

◆ ERDGBarrierLocation

enum class ERDGBarrierLocation : uint8
strong

Barrier location controls where the barrier is 'Ended' relative to the pass lambda being executed.

Most barrier locations are done in the prologue prior to the executing lambda. But certain cases like an aliasing discard operation need to be done after the pass being invoked. Therefore, when adding a transition the user can specify where to place the barrier.

Enumerator
Prologue 

The barrier occurs in the prologue of the pass (before execution).

Epilogue 

The barrier occurs in the epilogue of the pass (after execution).
