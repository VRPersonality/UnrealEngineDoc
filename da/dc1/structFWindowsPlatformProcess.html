Whether to expect to run at a low process integrity level or not.

This affects the paths that must be used for user and temp storage. The process may launch at default (medium) integrity level and then downgrade itself to low later for security benefits, so checking the current integrity level alone is not sufficient.

Returns
true if low integrity level should be expected, otherwise false.
