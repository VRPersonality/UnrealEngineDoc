27/**
28 * Proxy class that represents scene instance data to the renderer.
29 * Responsible for preparing data for render use per-platform & serializing such data for cooked builds.
30 * Supplies a persistent ID mapping for use in FScene. The ID mapping is reset when the proxy is recreated.
31 * Thus the FScene will generate full remove/add handling for proxy recreate events.
32 * Not responsible for storing the component representation of the instance data.
33 *
34 * TODO: Add concept of async update and needing to sync the proxy before use.
