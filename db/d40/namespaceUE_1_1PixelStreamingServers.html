PIXELSTREAMINGSERVERS_API TSharedPtr< IServer > UE::PixelStreamingServers::MakeCirrusServer ( )

Creates a NodeJS Cirrus signalling server by launching it as a child process.

Note: Calling this method does not launch the server. You should call Launch() yourself once you have bound to appropriate delegates such as OnReady.

Returns
The Cirrus signalling server wrapped in a server object for simple management.

Creates a native C++ signalling server (matching signalling functionality of cirrus.js from legacy versions, pre UE5) with no dependencies launched inside the Unreal Engine process.
