Detailed Description

template<typename BaseInterface>
class TPackageWriterToSharedBuffer< BaseInterface >

A base class for IPackageWriter subclasses that writes to records that are read in CommitPackage.

To avoid diamond inheritance, this class specifies the interface it is implementing (either IPackageWriter or ICookedPackageWriter) by template. Subclasses should derive from one of TPackageWriterToSharedBuffer<IPackageWriter> TPackageWriterToSharedBuffer<ICookedPackageWriter>.
