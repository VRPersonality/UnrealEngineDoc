Detailed Description

Enum used to define a entrance selection priority.

Highest priority is preferred, but when the priority is the same the selection method (distance) is used to decide which entrance is chosen. Annotation to define a entrance locations for a Smart Object Slot. This can be used to add multiple entry points to a slot, or to validate the entries against navigation data.
