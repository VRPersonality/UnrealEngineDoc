11/**
12 * Adapter class for iterator implementation.
13 *
14 * This class provides an interface to implement UE compatible iterators with a minimum set of implementation
15 * requirements. This provides the user with an easier way to implement custom iterators. The adapter uses a CRTP
16 * pattern requiring its base class to implement the following:
17 *
18 * Types:
19 * - ElementType: the element type used in the container.
20 * - SizeType: the type use for the container's element indexing and iterator's offset.
21 * Methods:
22 * - bool Equals(const IteratorType& other)
23 * - ElementType& Dereference() const;
24 * - void Increment();
25 * Optional:
26 * - void Decrement();
27 */
28template <typename Base>
