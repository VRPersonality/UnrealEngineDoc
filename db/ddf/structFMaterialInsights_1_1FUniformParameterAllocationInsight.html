Detailed Description

Wraps information about a uniform parameter (ScalarParametr, VectorParameter) allocation in the uniform expression data buffer.

An instance of this struct specifies wheere the components of a uniform parameter in the material were allocated in the preshader uniform buffer, that is which vec4 slot and which target components of that vec4.
