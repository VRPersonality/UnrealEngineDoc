Detailed Description

Indicate what types of resources should be included for calculating used memory.

Enumeration Type Documentation

◆ Type

enum EResourceSizeMode::Type
Enumerator
Exclusive 

Only include memory used by non-UObject resources that are directly owned by this UObject.

This is used to show memory actually used at runtime

EstimatedTotal 

Include exclusive resources and UObject serialized memory for this and all child UObjects, but not memory for external referenced assets or editor only members.

This is used in the editor to estimate maximum required memory
