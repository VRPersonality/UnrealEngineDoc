TMap<FIoChunkId, FIoHash> FAssetPackageData::ChunkHashes

The hash of all the chunks written by the package writer for this package.

This is only available after cooking. (Note, this only includes the ExportBundleData chunk if using ZenStoreWriter, as that chunk contains both the uasset and uexp data, which is managed separately in LooseCookedPackageWriter.)
