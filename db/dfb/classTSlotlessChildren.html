Detailed Description

template<typename ChildType>
class TSlotlessChildren< ChildType >

Some advanced widgets contain no layout information, and do not require slots.

Those widgets may wish to store a specialized type of child widget. In those cases, using TSlotlessChildren is convenient.

TSlotlessChildren should not be used for general-purpose widgets.
