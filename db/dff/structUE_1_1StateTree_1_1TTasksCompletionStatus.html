template<typename T>
ETaskCompletionStatus UE::StateTree::TTasksCompletionStatus< T >::GetCompletionStatus ( ) const
inlinenodiscard

The completion status of all tasks or any task, in priority order.

If any of the tasks fail, the completion status will be Failed regardless of any other tasks. In case the type is Any, it will return Succeeded before Stopped and before Running.

Returns
the completion status of all tasks.
