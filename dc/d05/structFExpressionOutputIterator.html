Detailed Description

An iterator for traversing the outputs of a UMaterialExpression.

This struct provides a way to iterate over the outputs of a given UMaterialExpression. It starts from the first output and advances through subsequent outputs until all are iterated through.

Usage:

UMaterialExpression* MyExpression = ...;
for (FExpressionOutputIterator It{ MyExpression }; It; ++It)
{
// Process the output, e.g. It->Output->IsConnected();
}
