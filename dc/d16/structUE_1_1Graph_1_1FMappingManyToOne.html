Detailed Description

Represents a mapping between two graphs where each vertex in a source graph maps to exactly one vertex in a target graph.

The mapping is not bidirectionally one-to-one. The length of Mapping is the number of vertices in the source graph and each value is a vertex in the target graph. e.g. a mapping from the vertices in a cyclic graph to the vertices in its condensation graph.
