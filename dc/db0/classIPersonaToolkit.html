◆ SetPreviewMesh()

virtual void IPersonaToolkit::SetPreviewMesh ( USkeletalMesh * InSkeletalMesh,
bool bSetPreviewMeshInAsset = true )
pure virtual

Set the preview mesh, according to context (mesh, skeleton or animation etc.) Note that this sets the mesh in the asset and in the viewport (and may re-open the asset editor to apply this).

Parameters
InSkeletalMeshThe mesh to set
bSetPreviewMeshInAssetIf true, the mesh will be written to the asset so it can be permanently saved. Otherwise the change is merely transient and will reset next time the editor is opened.
