Detailed Description

template<typename DeleteTaskType>
class UE::Geometry::TDeleterTask< DeleteTaskType >

TDeleterTask is a task that takes ownership of another FAsyncTask that is (presumably) still executing some operation.

The DeleterTask waits for this FAsyncTask to complete, and then deletes it.

TDeleterTask should be launched as a background task via FAutoDeleteAsyncTask, see FAsyncTaskExecuterWithAbort::CancelAndDelete for an example.
