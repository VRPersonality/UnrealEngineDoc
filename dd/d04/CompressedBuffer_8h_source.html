39/**

203/**
204 * A type that stores the state needed to decompress a compressed buffer.
205 *
206 * The compressed buffer can be in memory or can be loaded from a seekable archive.
207 *
208 * The reader can be reused across multiple source buffers, which allows its temporary buffers to
209 * be reused if they are the right size.
210 *
211 * It is only safe to use the reader from one thread at a time.

213 * @see FCompressedBuffer
