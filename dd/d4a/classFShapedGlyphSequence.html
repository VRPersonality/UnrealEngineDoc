enum class FShapedGlyphSequence::EEnumerateGlyphsResult : uint8
strongprivate

Enumerate all of the glyphs within the given source index range (enumerates either visually or logically)

Note
The indices used here are relative to the start of the text we were shaped from, even if we were only shaped from a sub-section of that text
Returns
EnumerationComplete if we found the start and end point and enumerated the glyphs, EnumerationAborted if the callback returned false, or EnumerationFailed (eg, because you started or ended on a merged ligature, or because the range is out-of-bounds)
