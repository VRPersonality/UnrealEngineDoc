Detailed Description

template<typename T>
struct UE::StructuredArchive::Private::TOptionalNamedAttribute< T >

Class to contain a named attribute for serialization, with a default.

Intended to be created as a temporary and passed to object serialization methods, which can choose not to serialize the attribute if it matches the default.
