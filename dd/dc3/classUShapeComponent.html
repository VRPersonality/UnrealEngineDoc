◆ bUseArchetypeBodySetup

UPROPERTY () ) uint8 bUseSystemDefaultObstacleAreaClass uint8 UShapeComponent::bUseArchetypeBodySetup
protected

Only show this component if the actor is selected.

If true it allows Collision when placing even if collision is not enabled If set, shape will be exported for navigation as dynamic modifier instead of using regular collision data Uses FNavigationSystem::GetDefaultObstacleArea() by default instead of AreaClassOverride, bDynamicObstacle must be true to use this. If the body setup can be shared (i.e. there have been no alterations compared to the CDO)
