18/**
19 * A Reply is something that a Slate event returns to the system to notify it about certain aspect of how an event was handled.
20 * For example, a widget may handle an OnMouseDown event by asking the system to give mouse capture to a specific Widget.
21 * To do this, return FReply::CaptureMouse( NewMouseCapture ).
22 */
23class FReply : public TReplyBase<FReply>
