Detailed Description

The IPixelStreaming2Module interface manages the core functionality of the Pixel Streaming system.

This class provides access to streamers, video and audio producers, and handles starting and stopping the streaming process. It also allows interaction with the signalling server and manages the lifecycle of streamers within the Pixel Streaming system.
