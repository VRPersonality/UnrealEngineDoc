FNodeMatch FGraphDiffControl::FindNodeMatch ( UEdGraph * OldGraph,
UEdGraphNode * NewNode,
TArray< FNodeMatch > const & PriorMatches )
static

Looks through the supplied graph for a node that best matches the one specified.

Sometimes (when diffing separate assets) there could be more than one possible match, so providing a list of already matched nodes helps us narrow it down (and prevents us from matching one node with multiple others).

Parameters
OldGraphThe graph you want to search.
NewNodeThe new node you want to match.
PriorMatchesPrevious made matches to exclude from our search.
Returns
A pair of nodes (including the supplied one) that best match each other (one may be nullptr if no match was found).
