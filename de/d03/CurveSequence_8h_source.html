13/**
14 * A sequence of curves that can be used to drive animations for slate widgets.
15 * Active timer registration is handled for the widget being animated when calling play.
16 *
17 * Each curve within the sequence has a time offset and a duration.
18 * This makes FCurveSequence convenient for crating staggered animations.
19 * e.g.
20 * // We want to zoom in a widget, and then fade in its contents.
21 * FCurveHandle ZoomCurve = Sequence.AddCurve( 0, 0.15f );
22 * FCurveHandle FadeCurve = Sequence.AddCurve( 0.15f, 0.1f );
23 * Sequence.Play( this->AsShared() );
24 */
