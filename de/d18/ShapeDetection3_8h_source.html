17 /**
18 * Detect if input Mesh is a meshed approximation of an analytic Sphere, and if so return best guess in SphereOut.
19 * Fits a sphere to input points with several rounds of incremental improvement, then measures chordal deviation of edge midpoints.
20 * @param RelativeDeviationTol Scaled by sphere diameter. The allowed difference in the distance from edge midpoints to the surface of the sphere vs the ideal distance for an edge of the same length.
21 * @param MaxAngleRangeDegrees Maximum angle difference in vectors from center of sphere to the endpoints of any surface edge. Controls how coarsely tessellated a sphere can be before it is not considered a sphere.
22 * @return true if mesh is a Sphere and SphereOut is initialized
23 */
24 bool DYNAMICMESH_API IsSphereMesh(const FDynamicMesh3& Mesh, FSphere3d& SphereOut, double RelativeDeviationTol = 0.025, double MaxAngleRangeDegrees = 72);

27 /**
28 * Detect if input Mesh is a meshed box, and if so return analytic box in BoxOut.
29 * Clusters face planes, looking to find 6 unique planes with normals grouped into 3 opposite-direction pairs.
30 * If this configuraiton is found, computing minimal box is trivial.
31 * @param AngleToleranceDeg normals are allowed to deviate by this amount and still be considered coplanar
32 * @param PlaneDistanceTolerance planes with the same normal are allowed to be this far apart and still be considered coplanar
33 * @return true if mesh is a Box and BoxOut is initialized
34 */
35 bool DYNAMICMESH_API IsBoxMesh(const FDynamicMesh3& Mesh, FOrientedBox3d& BoxOut, double AngleToleranceDeg = 0.1, double PlaneDistanceTolerance = 1e-2);

40 * Fits a capsule to input points, then measures chordal deviation of edge midpoints.
41 * @param RelativeDeviationTol Scaled by capsule diameter. The allowed difference in the distance from edge midpoints to the surface of the capsule vs the ideal distance for an edge of the same length.
42 * @param MaxAngleRangeDegrees Maximum angle range an edge can cover over the cylinder or the spherical endcaps. Controls how coarsely tessellated a capsule can be before it is not considered a capsule.

44 */
