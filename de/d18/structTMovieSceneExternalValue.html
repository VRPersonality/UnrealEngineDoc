Detailed Description

template<typename T>
struct TMovieSceneExternalValue< T >

Typed external value that can be used to define how to access the current value on an object for any given channel of data.

Typically defined as the extended editor data for many channel types through TMovieSceneChannelTraits::ExtendedEditorDataType.
