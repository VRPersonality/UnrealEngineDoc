◆ Source

FDynamicSpriteEmitterReplayData FDynamicSpriteEmitterData::Source

The frame source data for this particle system.

This is everything needed to represent this this particle system frame. It does not include any transient rendering thread data. Also, for non-simulating 'replay' particle systems, this data may have come straight from disk!
