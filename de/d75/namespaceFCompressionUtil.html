Logs or writes to an array of strings a memory region as a hex dump For logging, log category: LogCore, level: Display.

Bytes, BytesNum are used to designate the overall memory region of interest (e.g. a file in memory), but only bytes from OffsetStart to OffsetEnd will be logged out, and current offset will be printed at each line's start.

Parameters
Bytesmemory location of the data
BytesNumtotal size of the data
OffsetStartoffset to first byte to be logged out (pass 0 if you want to log out whole region)
OffsetEndoffset to the end of log window (pass BytesNum if you want to log out whole region)
BytesPerLinehow many bytes to print per line of hex dump
