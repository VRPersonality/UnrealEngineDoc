Detailed Description

template<typename RealType>
class UE::Geometry::TIntersector1< RealType >

TIntersector1 computes the intersection of one-dimensional intervals [u0,u1] and [v0,v1].

The end points of the input intervals must be ordered u0 <= u1 and v0 <= v1. Infinite and degenerate intervals are allowed.
