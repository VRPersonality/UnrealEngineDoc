◆ PrintWorkerMemoryUsage()

virtual bool FShaderCompileThreadRunnableBase::PrintWorkerMemoryUsage ( bool bAllowToWaitForLock = true)
inlineprotectedvirtual

Tries to print out the memory usage of all shader compile workers.

When called during an out-of-memory event, it is useful to allow this process to wait for any locks, so we can rule out deadlocks while reporting out-of-memory errors. Returns whether the memory usage was successfully printed.

◆ WorkerStateHeartbeat()

bool FShaderCompileThreadRunnableBase::WorkerStateHeartbeat ( uint64 InWorkerStateHash)
protected

Returns false if the new worker state hash hasn't changed in a certain period of time.

If false, a warning is printed. An input hash of 0 will always succeed and effectively reset the timer. A value of 0 should be used when no jobs are pending.
