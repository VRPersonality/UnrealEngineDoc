Detailed Description

template<typename ValueType, typename CustomDataType>
class TCustomIndexedValuesChange< ValueType, CustomDataType >

TCustomIndexedValuesChange is an implementation of TIndexedValuesChange that allows you to provide the Apply() And Revert() implementations via lambdas, as well as specify a custom data element to be stored with the Change.

For example if this is a change to an attribute array, you could store the attribute identifier in the CustomData.
