25/*
26 * A struct that can reference a top level asset such as '/Path/To/Package.AssetName'
27 * Stores two FNames internally to avoid
28 * a) storing a concatenated FName that bloats global FName storage
29 * b) storing an empty FString for a subobject path as FSoftObjectPath allows
30 * Can also be used to reference the package itself in which case the second name is NAME_None
31 * and the object resolves to the string `/Path/To/Package`
32 * This struct is mirrored and exposed to the UE reflection system in NoExportTypes.h
33*/
