Detailed Description

MovieScene evaluation context.

Should remain bitwise copyable, and contain no external state since this has the potential to be used on a thread

◆ GetTraversedFrameNumberRange()

MOVIESCENE_API TRange< FFrameNumber > FMovieSceneEvaluationRange::GetTraversedFrameNumberRange ( ) const

Get the range of frame numbers traversed over this evaluation range by flooring the lower bound, and ceiling the upper bound.

For example: a time range of [1.5, 5.6] will yield the equivalent of [1, 6). A time range of (2.0, 2.9) will yield the equivalent of [2,3).
