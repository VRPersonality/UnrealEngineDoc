◆ FastReleaseExcessVoices()

HARMONIXDSP_API uint32 FFusionVoicePool::FastReleaseExcessVoices ( FFusionSampler * InSampler = nullptr)

Fast releases voices that have exceeded the soft limit for the pool (or the channel).

If no channel is specified, then voices assigned to any channel are candidates for being fast-released. If a specific channel is specified, then only voices being used by that channel may be fast released.

Parameters
channel(optional) restrict the voice candidates for release to the ones uses by a specific channel.
Returns
the number of voices currently being used by the pool (or by the specified channel)
