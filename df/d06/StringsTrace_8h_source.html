26 * Utility class to trace deduplicated strings, FNames and static strings. Each function returning a
27 * reference that can be used when tracing events.
28 *
29 * A event defined as:
30 * \code
31 * UE_TRACE_EVENT_BEGIN(Asset)
32 * UE_TRACE_EVENT_REFERENCE_FIELD(Strings, FName, Name)
33 * UE_TRACE_EVENT_REFERENCE_FIELD(Strings, StaticString, File)
34 * UE_TRACE_EVENT_END()
35 *
36 * Can be emitted with:
37 * auto NameRef = FStringTrace::GetNameRef(Object->GetFName());
38 * auto FileRef = FStringTrace::GetStaticStringRef(__FILE__);
39 * UE_TRACE_LOG(Asset)
40 * << Asset.Name(NameRef)
41 * << Asset.File(FileRef);
42 * \endcode
43 */
