25/**
26 * The base class for custom replaceable bindings. A replaceable binding uses an internal custom spawnable at editor time to produce a preview object,
27 * while in editor will use some other mechanism to dynamically bind an object to the track. Different replaceable types can choose different combinations
28 * of how to create a spawnable for preview vs. how to dynamically bind an object at runtime.
29 * UMovieSceneReplaceableActorBinding as an example is the simplest type of replaceable binding and provides no method for binding at runtime and relies on the LevelSequenceActor's Binding Override
30 * mechanism to bind an actor at runtime.
31 */
