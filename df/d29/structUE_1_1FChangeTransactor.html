Detailed Description

Helper structure used for inserting FChange based undo/redo operations inside of the GUndo (ITransaction) instance.

The operations are targeted a specific UObject, which is used as the 'transactional' object. The transactions themselves are initiated using the ITransactor API using GEngine as the instance.
