ANIMNEXT_API TWeakPtr< FRigVMRuntimeData > UE::AnimNext::FRigVMRuntimeDataRegistry::FindOrAddLocalRuntimeData ( const FRigVMRuntimeDataID & RigVMRuntimeDataID,
const FRigVMExtendedExecuteContext & ReferenceContext )
static

Finds or adds one VM runtime data instance for the passed ID in the TLS.

If it exists and the Context hash is correct, returns the TLS weak pointer. If not found or the context hash is not correct, it creates a new instance in the Global storage and adds a Weak pointer to the TLS.

Parameters
RigVMRuntimeDataIDThe key to find or associate (if created) the instance with.
ReferenceContextReference context to copy data from (if not found or found but the context hash is not correct)
bOutHasBeenUpdatedOut : True If the ID did not exist or existed but the Context hash was not correct
Returns
A weak pointer to the data as stored in the TLS map.
