bool FPaths::CollapseRelativeDirectories ( FString & InPath)
static

Takes a fully pathed string and eliminates relative pathing (eg: annihilates ".." with the adjacent directory).

Assumes all slashes have been converted to TEXT('/'). For example, takes the string: BaseDirectory/SomeDirectory/../SomeOtherDirectory/Filename.ext and converts it to: BaseDirectory/SomeOtherDirectory/Filename.ext
