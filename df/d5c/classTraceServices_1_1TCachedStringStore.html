Detailed Description

template<typename CharType>
class TraceServices::TCachedStringStore< CharType >

A deduplicating persistent string store.

Allows users to store a string in a session and retrieve it at a given offset in consecutive sessions. The store uses analysis cache to store the strings. The store is not thread safe.
