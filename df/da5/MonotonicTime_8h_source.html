64/**
65 * A point in time measured in seconds since an arbitrary epoch.
66 *
67 * This is a monotonic clock which means the current time will never decrease. This time is meant
68 * primarily for measuring intervals. The interval between ticks of this clock is constant except
69 * for the time that the system is suspended on certain platforms. The tick frequency will differ
70 * between platforms, and must not be used as a means of communicating time without communicating
71 * the tick frequency together with the time.
72 */
