7// Generic associative container for instance data keyed off of the FNetworkPredictionID
8// SortedMap + SparseArray provides:
9// -O(log n) add/remove.
10// -Stable index that can be cached for O(1) lookup in various service acceleration structures
11// -Data* are not stable out of this data structure
