KISMET_API const UK2Node * FBlueprintActionMenuUtils::ExtractNodeTemplateFromAction ( const FEdGraphSchemaAction & PaletteAction)
static

A number of different palette actions hold onto node-templates in different ways.

This handles most of those cases and looks to extract said node- template from the specified action.

Parameters
PaletteActionThe action you want a node-template for.
Returns
A pointer to the extracted node (NULL if the action doesn't have one, or we don't support the specific action type yet)
