bool UE::MovieScene::FEntityGroupID::HasGroup ( ) const
inline

Returns whether this component is valid (see IsValid) and if it still belongs to a group.

If the entity was removed from its group, the GroupIndex would still be valid, but we would have set Flags to RemovedFromGroup. This lets downstream systems still know what group the entity used to be part of, for their own book-keeping.
