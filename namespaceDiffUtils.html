UNREALED_API bool DiffUtils::Identical ( const FResolvedProperty & AProp,
const FResolvedProperty & BProp,
const UObject * OwningOuterA,
const UObject * OwningOuterB )

DiffUtils now requires root objects so that object topology can be meaningfully compared.

DiffUtils::Identical works similar to FProperty::Identical except when a UObject is found, that is in OwningOuter*, it's compared by topology instead. This allows sub-objects to diff correctly.
