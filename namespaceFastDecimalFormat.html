Detailed Description

Provides efficient and culture aware number formatting and parsing.

You would call FastDecimalFormat::NumberToString to convert a number to the correct decimal representation based on the given formatting rules and options. You would call FastDecimalFormat::StringToNumber to convert a string containing a culture correct decimal representation of a number into an actual number. The primary consumer of this is FText, however you can use it for other things. GetCultureAgnosticFormattingRules can provide formatting rules for cases where you don't care about culture.

Note
If you use the version of FastDecimalFormat::NumberToString that takes an output string, the formatted number will be appended to the existing contents of the string.
