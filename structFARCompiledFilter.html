Detailed Description

A struct to serve as a filter for Asset Registry queries.

Each component element is processed as an 'OR' operation while all the components are processed together as an 'AND' operation. This is a version of FARFilter optimized for querying, and can be generated from an FARFilter by calling IAssetRegistry::CompileFilter to resolve any recursion.
