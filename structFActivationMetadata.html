Detailed Description

Metadata that can be set on activatable widgets via GetActivationMetadata & listened to via OnActivationMetadataChanged.

Useful for game-specific behaviors to be triggered by activation of certain widgets generically. For example changing camera config of your game when certain widgets activate.

By default only has an enum for this metadata.
