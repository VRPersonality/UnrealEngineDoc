◆ InterpolationOverride

TOptional<EAnimInterpolationType> FAnimExtractContext::InterpolationOverride

The optional interpolation mode override.

If not set, it will simply use the interpolation mode provided by the asset. One example where this could be used is if you want to force sampling the animation with Step interpolation even when the animation sequence asset is set to Linear interpolation.
