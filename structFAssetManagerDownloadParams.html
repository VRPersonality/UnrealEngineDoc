Detailed Description

EXPERIMENTAL - Additional parameters for downloading packages Asset Manager will automatically create a FStreamableDownloadCachePin for each primary asset requested with download params.

Cache pins can be retrieved with FStreamableHandle::GetDownloadCachePins() Cache pins will be released along with the primary asset or when bundle state changes.
