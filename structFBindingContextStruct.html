Detailed Description

Binding context struct allow to describe information for a struct to bind to using the binding widget.

An array of structs is passed to the widget to describe the context in which the binding exists. When the widget selectes a property from binding context struct array, the first FBindingChainElement's index correlates to the array passed to the widget.
