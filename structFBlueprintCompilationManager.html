void FBlueprintCompilationManager::RegisterCompilerExtension ( TSubclassOf< UBlueprint > BlueprintType,
UBlueprintCompilerExtension * Extension )
static

Registers a blueprint compiler extension - anytime a blueprint of the provided type is compiled the extension will be activated.

Note that because editor initialization may require blueprint compilation there may be blueprints compiled before the extension is registed unless special care has been taken.

References BlueprintType.

◆ ReparentHierarchies()

void FBlueprintCompilationManager::ReparentHierarchies ( const TMap< UClass *, UClass * > & OldClassToNewClass)
static

Safely reparents all child classes of every Key in OldClassToNewClass to the class in the corresponding Value.

Typically this means every child type will be reinstanced - although reinstancing could be avoided when layouts match.
