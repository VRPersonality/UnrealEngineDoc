Detailed Description

Structure to store the buffer offsets to the section's cloth deformer mapping data.

Also contains the stride to further cloth LODs data for cases where one section has multiple mappings so that it can be wrap deformed with cloth data from a different LOD. When using LOD bias in Raytracing for example.
