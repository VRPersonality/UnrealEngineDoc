void FCollisionQueryParams::SetNumIgnoredComponents ( int32 NewNum)

Set the number of ignored components in the list.

Uniqueness is not changed, it operates on the current state (unique or not). Useful for temporarily adding some, then restoring to a previous size. NewNum must be <= number of current components for there to be any effect.
