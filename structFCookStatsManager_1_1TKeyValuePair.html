Detailed Description

template<typename KeyType, typename ValueType>
struct FCookStatsManager::TKeyValuePair< KeyType, ValueType >

Copy of TKeyValuePair<> from Core, but with a default initializer for each member.

Useful for containers that auto-add default constructed members. We can't change TKeyValuePair because code relies on it and don't want to slow it down for those use cases.
