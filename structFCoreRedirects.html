◆ AddAssetRedirects()

void FCoreRedirects::AddAssetRedirects ( const TMap< FSoftObjectPath, FSoftObjectPath > & InRedirects)
static

Adds a collection of redirects as Type_Asset.

These allow FCoreRedirects to support the functions of UObjectRedirector. Any duplicate sources are logged and discarded (only the first redirect from a path is used) Package redirects corresponding to the soft object paths are implicitly created.
