bool FDetailedTickStats::BeginObject ( UObject * Object)

Starts tracking an object and returns whether it's a recursive call or not.

If it is recursive the function will return false and EndObject should not be called on the object.

Parameters
ObjectObject to track
Returns
false if object is already tracked and EndObject should NOT be called, true otherwise
