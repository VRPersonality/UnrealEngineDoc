◆ bAllowCollapsingInternalEdgeWithBoundaryVertices

bool UE::Geometry::FDynamicMesh3::FCollapseEdgeOptions::bAllowCollapsingInternalEdgeWithBoundaryVertices = false

When false, collapse is disallowed if the edge is an interior edge, yet both vertices are connected to boundary edges.

In some circumstances this could create a bowtie. In other cases, it could disconnected parts of a mesh that were connected by a bowtie.

◆ bAllowHoleCollapse

bool UE::Geometry::FDynamicMesh3::FCollapseEdgeOptions::bAllowHoleCollapse = false

When false, collapse is disallowed if the edge is the boundary of a single triangle hole, such that collapsing it would clse the hole.

I.e. collapse is dissallowed if there is some vertex that connects vKeep and vRemove that is not part of the triangle(s) being collapsed (note that if such a vertex is connected by a non-boundary edge, collapse will always be disallowed regardless of bAllowHoleCollapse, as it would create non-manifold geometry).

◆ bAllowTetrahedronCollapse

bool UE::Geometry::FDynamicMesh3::FCollapseEdgeOptions::bAllowTetrahedronCollapse = false

When false, collapse is disallowed if we are collapsing the side of a tetrahedron.

Note that a base edge of an open-base tetrahedron could be collapsed even if bAllowTetrahedronCollapse is false if bAllowHoleCollapse is true.
