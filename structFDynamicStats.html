template<typename TStatGroup>
TStatId FDynamicStats::CreateStatId ( const FString & StatNameOrDescription)
inlinestatic

Create a new stat id and registers it with the stats system.

This is the only way to create dynamic stat ids at runtime. Can be used only with FScopeCycleCounters.

Store the created stat id. Expensive method, avoid calling that method every frame.

Example: FDynamicStats::CreateStatId<STAT_GROUP_TO_FStatGroup( STATGROUP_UObjects )>( FString::Printf(TEXT("MyDynamicStat_%i"),Index) )
