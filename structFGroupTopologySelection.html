These are helper methods to get out a value when you know you only have one, or you don't care which one of the existing ones you grab.

They only exist because TSet doesn't have a clean way to grab one arbitrary element besides *TSet<int32>TConstIterator(MySet). The hope is that these are slightly cleaner.
