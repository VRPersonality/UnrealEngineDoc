TArray<int8, TInlineAllocator<MAX_MESH_LOD_COUNT> > FMeshDeformerSet::DeformerIndexForLOD

Indexed by mesh LOD.

Each element is either a valid index into the Deformers array, or INDEX_NONE to signify that no deformer should be used for this LOD. MAX_MESH_LOD_COUNT here cannot be a hard limit as editor allows more LODs to be created however there may be other systems assuming MAX_MESH_LOD_COUNT is respected so generally it is not a good idea to exceed it
