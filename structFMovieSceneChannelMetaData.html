◆ SubPropertyPathMap

TMap<FName, FName> FMovieSceneChannelMetaData::SubPropertyPathMap

Path representation of a sub-property relative to a class property (i.e.

topmost property) but NOT including the topmost property itself. This should be used when a Channel can be used by multiple struct sources and happen to have different property names for these. The prime example of this is FTransform vs FEulerTransform. FTransform uses "Translation" vs FEulerTransform uses "Location"
