static

Parse the next space-delimited (or SingleCharacterDelmiter-delimited) expression, using brackets (), {}, or [] to allow subexpressions to be part of the single expression even if they contain the delimiter.

The parsed expression can be either an atomic token such as "Value1" in the stream "Value1 Value2 Value3" or a bracketed expression such as "(Value1.1 Value1.2)" in the stream "(Value1.1 Value1.2) Value2 Value3". If quotes are encountered, text inside of the quotes is not interpreted (brackets and delimiters are ignored) and it is included verbatim in the parsed expression. Trailing delimiters are not consumed, only leading delimiters.
