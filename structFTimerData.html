◆ bMaxOncePerFrame

uint8 FTimerData::bMaxOncePerFrame

If true for a looping timer, it will run a maximum of once per frame when it expires.

Otherwise, it will run as many times as can fit in the current frame's delta time (rounding down).

◆ ExpireTime

double FTimerData::ExpireTime

Time (on the FTimerManager's clock) that this timer should expire and fire its delegate.

Note when a timer is paused, we re-base ExpireTime to be relative to 0 instead of the running clock, meaning ExpireTime contains the remaining time until fire.
