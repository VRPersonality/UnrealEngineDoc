Detailed Description

template<typename InElementType>
struct TContainerElementTypeCompatibility< InElementType >

Temporary compatibility mechanism to be used solely for the purpose of raw pointers to wrapped pointers.

Specialization and use of this type is not supported beyond the specific instances in ObjectPtr.h. These types and the container methods that leverage them will be removed in a future release.
