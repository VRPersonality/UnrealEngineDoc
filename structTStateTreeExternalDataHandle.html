Detailed Description

template<typename T, EStateTreeExternalDataRequirement Req = EStateTreeExternalDataRequirement::Required>
struct TStateTreeExternalDataHandle< T, Req >

Handle to access an external struct or object.

This reference handle can be used in StateTree tasks and evaluators to have quick access to external data. The type provided to the template is used by the linker and context to pass along the type.

USTRUCT() struct FExampleTask : public FStateTreeTaskBase { ...

bool Link(FStateTreeLinker& Linker) { Linker.LinkExternalData(ExampleSubsystemHandle); return true; }
