Detailed Description

Node Description A node description represents a unique instance in the authored static graph.

A node description may have any number of runtime instances in the dynamically executed graph. As such, a node description is read-only at runtime while a node instance is read/write.

A node description is followed in memory by the trait descriptions (their shared read-only data) that live within it. Trait descriptions include things like hard-coded/inline properties, pin links, etc.

A node description is itself an instance of a node template.
