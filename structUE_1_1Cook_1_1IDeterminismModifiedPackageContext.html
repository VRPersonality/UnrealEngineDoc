◆ GetExportContext()

virtual const IDeterminismModifiedExportContext & UE::Cook::IDeterminismModifiedPackageContext::GetExportContext ( UObject * Export)
pure virtual

Get the IDeterminismModifiedExportContext object for the given export.

Creates it if it doesn't exist; if it doesn't exist the IDeterminismModifiedExportContext will have IsModified=false and emtpy diagnostics. Invalid to call with a Export pointer that is not in the package; in that case it will log an error and return an empty IDeterminismModifiedExportContext pointing to the UPackage itself.
