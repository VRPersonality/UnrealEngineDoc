Detailed Description

FGeometrySelection represents a subset of geometric elements of a larger object, for example a Mesh (currently the only use case).

The main selection is represented via 64-bit unsigned integers. The integers are stored in a TSet for efficient unique adds and removals. No assumptions are made about the values, they could be (eg) mesh indices, IDs of some type, or even pointer values.
