Detailed Description

Represents a mapping between two graphs where each vertex in a source graph maps to multiple vertices in a target graph.

The length of Mapping is the number of vertices in the source graph and the contents of each mapping are vertices in the target graph. Buffer stores the contents of all array views in Mapping. e.g. a mapping between the vertices in a condensation graph and the original cyclic graph it was generated from.
