Detailed Description

Struct passed to ILandscapeEditLayerRenderer's render functions.

This would normally be an inner struct of ILandscapeEditLayerRenderer but this prevents forward-declaration so we leave in the UE::Landscape::EditLayers namespace instead.

◆ RendererState

FEditLayerRendererState UE::Landscape::EditLayers::FRenderParams::RendererState

Full state for the renderer involved in this step.

This allows to retrieve the exact state of this renderer (e.g. enabled weightmaps, which can be different than the target layer group, in that target layers A, B and C might belong to the same group but this renderer actually only has A enabled). This is therefore the renderer's responsibility to check that a given target layer from the target layer group is effectively enabled.
