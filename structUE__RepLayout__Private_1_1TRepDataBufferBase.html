Detailed Description

template<ERepDataBufferType DataType, typename ConstOrNotType>
struct UE_RepLayout_Private::TRepDataBufferBase< DataType, ConstOrNotType >

TRepDataBuffer and TConstRepDataBuffer act as wrapper around internal data buffers that FRepLayout may use.

This allows FRepLayout to properly interact with memory buffers and apply commands to them more easily.
