◆ MaterialBasePath

FName UnFbx::FBXImportOptions::MaterialBasePath

This allow to add a prefix to the material name when unreal material get created.


This prefix can just modify the name of the asset for materials (i.e. TEXT("Mat")) This prefix can modify the package path for materials (i.e. TEXT("/Materials/")). Or both (i.e. TEXT("/Materials/Mat"))
